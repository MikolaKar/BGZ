Как работать с задачами из списка, не открывая их карточки

С задачами на закладке Задачи мне удобно работать, используя область просмотра, без открытия карточки в отдельном окне.

Чтобы включить область просмотра, выполните команду Показать область просмотра задачи из меню Все действия.

Кроме того, некоторые действия удобно выполнять из контекстного меню в списке задач: например, просматривать предмет или процесс, принимать к исполнению или помечать задачу флагами.

Как работать с предметами задач

Задача может содержать предметы - ссылки на различные объекты программы (например, документы, мероприятия или файлы) и файлы с диска.

Предметы выступают в качестве основания задачи или содержат дополнительные данные, помогающие в ее выполнении.

При включенной области просмотра с предметами можно работать, не открывая карточку задачи.

Добавление

При помощи команды  к задаче можно добавлять вспомогательные предметы. Такие предметы обозначаются  и содержат дополнительные данные, помогающие при выполнении задачи.

Заполнение

Предмет задачи, обозначенный  и выделенный красным цветом, называется заполняемым. Это значит, что его обязательно нужно заполнить перед выполнением задачи, иначе ее невозможно будет завершить.

Для заполнения предмета дважды щелкните по нему мышью и затем выберите нужный файл или объект.

Удаление

При помощи команды  исполнитель задачи может удалить те предметы, которые он добавил. Этой же командой он может очистить (но не удалить) заполняемый предмет.

Как искать в списке задач

В списке Задачи мне по умолчанию доступен поиск в списке.

Если необходимо выполнить поиск в динамическом списке, воспользуйтесь командой Показать выполненные из меню Все действия. После этого на командной панели станет доступна команда  Найти.

Для закладки Задачи от меня доступен поиск только по динамическим спискам.

Как сортировать задачи

В списке Задачи мне по умолчанию сортировка выполняется при помощи команды контекстного меню Сортировать по.

Если необходимо использовать сортировку по столбцам, выполните команду Показать выполненные из меню Все действия.

Для закладки Задачи от меня доступна только сортировка по столбцам.

Как посмотреть полный список задач

Полный список задач доступен по команде панели навигации Все задачи из раздела Совместная работа.

Кроме того, руководители подразделений могут просматривать задачи своих подчиненных по команде Задачи отдела из раздела Совместная работа.

Как помечать задачи флагами

Флаги можно устанавливать (и сбрасывать) при при помощи команды , расположенной на командной панели, или двойным щелчком мыши в столбце  списка задач.

Флаг по умолчанию устанавливается двойным щелчком мыши в столбце .

Флаг по умолчанию можно поменять в персональных настройках программы.

Задачи, помеченные флагами, можно сортировать по столбцу или группировать по флагу.

Флаги индивидуальны для каждого пользователя, т. е. пользователи видят только те флаги, которые установили сами.

Как задать вопрос по задаче и перенести ее срок

Вы можете отправить запрос на перенос срока или задать вопрос по задаче прямо из списка "Мои задачи".

Для этого в командной панели мини-карточки задачи предусмотрены кнопки:

 - для создания нового вопроса;

 - для отправки запроса на перенос срока.

Вопрос можно задать любому сотруднику. Запрос на перенос срока направляется автору задачи.

Если данные кнопки недоступны, скорее всего, вы забыли установить настройку программы Использовать рассмотрение вопросов выполнения задач.

Как отменить выполнение задач

Выполненную задачу можно вернуть на шаг назад кнопкой Отменить выполнение в карточке задачи. Для этого:

1. Откройте карточку задачи.

  • Удобнее всего — по ссылке из панели быстрого доступа,
  • Но можно и по команде Показать выполненные меню Все действия списка Мои задачи,
  • И через список История (если карточку задачи открывали совсем недавно). Чтобы отрыть список используйте сочетание клавиш Ctrl+Shift+H.

2. Нажмите кнопку Отменить выполнение и, если нужно, введите комментарий.

3. Задача вновь станет невыполненной. Если это была задача Согласовать, то ваша виза будет перенесена в историю.

Если кнопка Отменить выполнение скрыта, значит процесс перешел на следующий шаг выполнения и отменить задачу уже нельзя. В этом случае, свяжитесь с автором процесса и договоритесь об остановке или перезапуске процесса.

Отмена выполнения задач заносится в Историю событий задач.

Отмена выполнения возможна:

  • Если это задачи Исполнить и Согласовать;
  • Если отмену выполняет исполнитель задачи, его руководитель или делегат;
  • Если выбрано параллельное исполнение или согласование (Всем сразу или Смешано);
  • Если процесс не перешел дальше по маршруту.

Как поставить задачу на контроль

Задачу можно поставить на контроль:

  • следить за сроком и ходом выполнения работ,
  • переносить контрольный срок,
  • контролировать отдельных пользователей,
  • снимать задачу с контроля,
  • выводить аналитические отчеты,
  • смотреть, какие задачи находятся на контроле у ваших коллег.

Программа сама отследит истекающие и приближающиеся контрольные сроки и сообщит вам об этом по почте.

Чтобы поставить задачу на контроль, нажмите в командной панели карточки или списка Задачи от меня. При этом откроется Контрольная карточка, в которой нужно установить срок, указать контролируемое действие и исполнителя.

Если задача уже контролировалась ранее, кнопка откроет список всех контрольных карточек когда-либо созданных по задаче. В списке можно посмотреть, кто контролирует задачу и, при наличии прав, изменить параметры контроля.

После постановки на контроль рекомендуется:

  • Настроить уведомления об истекающих или пропущенных сроках (в настройках программы),
  • Следить за всеми контролируемыми задачами в окне На контроле.

    Как настроить автоматическое обновление

    По умолчанию список задач обновляется только вручную по нажатию кнопки  или клавиши F5.  

    Чтобы включить автообновление, выполните одноименную команду контекстного меню.

    Не рекомендуется устанавливать значения, близкие к минимальному (10 сек), т. к. это может замедлить работу программы.
