Как читать условные обозначения

Задачи могут иметь следующие обозначения:

Обозначение Что означает
Жирный шрифт / обычный шрифт Задача не принята к исполнению / принята к исполнению
Красный шрифт Задача не выполнена в срок
Флаги задач

В нижней части окна выводится статистическая информация о количестве задач:

  • Новых, т. е. непринятых к исполнению
  • Просроченных
  • Всего

Как посмотреть мои задачи

С личными задачами, полученными или назначенными, удобно работать при помощи отдельного списка, который доступен по команде панели навигации Мои задачи из раздела Совместная работа.

В частности в этом списке отражена важность задач, видны делегированные и ролевые задачи, а также есть возможность работать через область просмотра, не открывая карточку задачи.

Кроме этого, общий, неструктурированный список задач подчиненных доступен по команде панели навигации Все задачи из раздела Совместная работа.

Как работает быстрый поиск

Быстрый поиск служит для отбора задач по признаку завершенности, проекту или автору. Отбор ведется по содержимому выбранной папки.

Условия поиска указываются в нужном поле (для полей Проект и Автор достаточно ввести несколько первых букв). Результаты поиска выводятся в списке писем.

Поиск можно выключить, нажав кнопку с изображением лампочки. При этом критерии поиска сохранятся, а в списке отобразятся все задачи.

Как помечать задачи флагами

Флаги можно устанавливать (и сбрасывать) при при помощи команды , расположенной на командной панели, или двойным щелчком мыши в столбце  списка задач.

Флаг по умолчанию устанавливается двойным щелчком мыши в столбце .

Флаг по умолчанию можно поменять в персональных настройках программы.

Задачи, помеченные флагами, можно сортировать по столбцу или группировать по флагу.

Флаги индивидуальны для каждого пользователя, т. е. пользователи видят только те флаги, которые установили сами.

Как задать вопрос по задаче

Задать вопрос автору задачи или другими сотрудникам можно с помощью команды Задать вопрос в меню Все действия.

В открывшемся окне нужно ввести вопрос, установить его Важность, обозначить Срок ответа, прикрепить дополнительные файлы и посмотреть другие вопросы по данной задаче.

Отозвать вопрос можно командой Остановить в карточке вопроса. Повторно отправить на рассмотрение – кнопкой Уточнить.

Если команда для создания вопроса недоступна, скорее всего, вы забыли установить настройку программы Использовать рассмотрение вопросов выполнения задач.

Как настроить автоматическое обновление

По умолчанию список задач обновляется только вручную по нажатию кнопки  или клавиши F5.  

Чтобы включить автообновление, выполните одноименную команду контекстного меню.

Не рекомендуется устанавливать значения, близкие к минимальному (10 сек), т. к. это может замедлить работу программы.
