Выберите один из подходящих под текущие нужды шаблонов. Программа сгруппировала их в порядке, рекомендованном к использованию:

  • Рекомендованные шаблоны – шаблоны процессов, привязанные к определенному виду документов (список выводится только при создании процесса на основании документа).
  • Общие шаблоны – шаблоны процессов, которые пока не привязаны ни к одному из видов документов;
  • Новые процессы – создает карточку нового процесса по видам, заведенным в программе: согласование, исполнение, рассмотрение и т.д. Подходит для создания процессов с нуля, без использования шаблонов. При создании процесса На основании в списке отображаются только те процессы, которые подходят к контексту вашей работы. Например, процесс Регистрация нельзя стартовать на основании корреспондента или файла, поэтому в списке Новые процессы он представлен не будет.
  • Другие шаблоны – шаблоны процессов, которые настраивает ответственный за нормативно-справочную информацию. Если процесс создается из панели навигации раздела Совместная работа, в список попадут шаблоны, которые относятся хотя бы к одному из видов документов.

Чтобы открыть карточку шаблона процесса используйте клавишу F2.
