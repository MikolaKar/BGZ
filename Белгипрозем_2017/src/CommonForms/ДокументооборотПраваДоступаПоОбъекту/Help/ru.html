В данном окне можно:

  • Посмотреть, какие права рассчитаны для выбранного вами объекта (закладка Права).
  • Обновить права для данного объекта. Перерасчет выполняется немедленно, даже если права объекта стояли в очереди отложенного обновления.
  • Ознакомиться со списком настроек, которые были использованы при расчете (отобразятся на закладке Данные для расчета прав после обновления). 
  • Получить детальную информацию по настроенным правам из Дескриптор доступа. Информация предназначена для технического специалиста.

Обратите внимание: окно Права доступа служит только для просмотра настроек прав выбранного объекта, редактирование настроек в данном окне не предусмотрено.
