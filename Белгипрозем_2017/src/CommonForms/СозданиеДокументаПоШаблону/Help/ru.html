Форма создания документа по шаблону открывается, если для данного типа документов (Внутренний, Входящий или Исходящий) в системе заведены шаблоны.

При нажатии на кнопку Создать по шаблону или Заполнить по шаблону производится копирование реквизитов, файлов, списка личного доступа и категорий из шаблона документа в создаваемый документ. Кнопка Создать по шаблону доступна при создании документа, кнопка Заполнить по шаблону доступна при заполнении из шаблона существующего нового документа. 

При нажатии на кнопку Создать пустой происходит создание нового документа с незаполненными полями.

При нажатии на кнопку Отмена происходит прекращение создания нового документа.
