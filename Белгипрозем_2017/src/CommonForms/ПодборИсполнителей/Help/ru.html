Как подобрать нескольких исполнителей

Перед вами окно подбора нескольких исполнителей. В зависимости от контекста работы (для чего вы выбираете исполнителей) для подбора становятся доступны:

  • пользователи,
  • роли исполнителей,
  • функции автоподстановки,
  • корреспонденты,
  • личные адреса.

Если какой-либо из списков подбора отсутствует, значит в текущем контексте работы его значения не требуются. Например, автоподстановки доступны только при подборе исполнителей в шаблон документов или процессов и недоступны при подборе из карточек самих документов и процессов.

Какие значения автоподстановки использовать

При создании бизнес-процесса из шаблона программа автоматически вычисляет значение автоподстановки и подставляет его в бизнес-процесс.

Для автоподстановки исполнителей в шаблонах процессов обязательно дожен быть указан предмет. Именно его данные используются для подбора автоподстановок.

Значения автоподстановок могут быть привязаны к автору процесса, автору или ответственному за документ.  

Рекомендуем использовать привязку к ответственному за документ. Это может быть полезно, например, при увольнении или переводе автора документа в другое подразделение. В таком случае для актуализации прав на документ достаточно поменять только одно поле Ответственный.
