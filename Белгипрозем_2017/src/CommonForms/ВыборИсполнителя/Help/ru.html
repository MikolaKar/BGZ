Как выбрать исполнителя

В данном окне вы можете выбрать только одного исполнителя. Для того, чтобы добавить сразу нескольких исполнителей, воспользуйтесь кнопкой Подбор в том же списке исполнителей, откуда вызвали это окно.

В зависимости от контекста выбора (откуда вы открыли окно) в качестве исполнителя можно выбрать:

  • пользователя,
  • роль исполнителя,
  • функцию автоподстановки,
  • корреспондента,
  • личный адрес.

Если какой-либо из списков отсутствует, значит в текущем контексте выбора его значения не требуются. Например, автоподстановка доступна только при выборе исполнителя в шаблон документов или процессов и недоступна при выборе из карточек самих документов и процессов.

Какие значения автоподстановки использовать

При создании бизнес-процесса из шаблона программа автоматически вычисляет значение автоподстановки и подставляет его в бизнес-процесс.

Для автоподстановки исполнителей в шаблонах процессов обязательно дожен быть указан предмет. Именно его данные используются для подбора автоподстановок.

Значение автоподстановки может быть привязано к автору процесса, автору или ответственному за документ.  

Рекомендуем использовать привязку к ответственному за документ. Это может быть полезно, например, при увольнении или переводе автора документа в другое подразделение. В таком случае для актуализации прав на документ достаточно поменять только одно поле Ответственный.
