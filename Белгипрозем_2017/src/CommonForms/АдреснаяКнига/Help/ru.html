В адресной книге Вы можете выбрать одного или несколько получателей письма из сотрудников, корреспондентов, личных адресатов, списка Мои контакты, физических лиц (включить отображение списка можно с помощью команды  - флажок Физические лица)

Выбор получателя, группы пользователей или целого отдела возможен с помощью следующих способов:

  • двойной щелчок мыши;
  • перетаскивание мышью;
  • команда Переместить из списка выбранное.

Получателей, с которыми Вы чаще всего ведете переписку Вы можете добавить в список Мои контакты:

  1. Выделите получателя в списке Выбранные адресаты  или в списке сотрудников, корреспондентов, физических лиц.
  2. Выполните команду "Добавить в "Мои контакты""  в контекстном меню, пользователь будет добавлен в Мои контакты.
  3. Также в Мои контакты можно добавить группу пользователей, отдел, автосписок (командой «Добавить автосписок» в командной панели). Примеры автосписков: «Все руководители», «Все сотрудники моего подразделения», «Все пользователи информационной базы».

Чтобы быстро находить нужных получателей, используйте полнотекстовый поиск на закладке Поиск (закладка доступна только в клиент-серверном варианте информационной базы). Ниже представлены несколько примеров поиска:

  • Введите "Федоров" для поиска по фамилии;
  • "fedorov" - для поиска по email;
  • "бухгалтерия" - для поиска по подразделениям;
  • * - для поиска по части слова, например "Федо*".
