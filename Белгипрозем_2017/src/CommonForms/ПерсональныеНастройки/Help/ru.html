Для открытия формы персональных настроек следует в разделе Настройка и администрирование в панели действий в группе Сервис выполнить команду Персональные настройки.

Работа с файлами

Настройка Описание
Действие при выборе файла Указывает, какое действие выполнять при нажатии на клавишу Enter или двойном щелчке мыши на файле - открыть файл или открыть его карточку.
Выбор режима открытия при выборе файла Если установлена, то перед открытием файла откроется окно для выбора режима открытия - "просмотр" или "редактирование".
Показывать подсказки при редактировании файлов

Если установлена, то при работе через веб-клиент программа будет подсказывать о том, как удобнее открыть файл для просмотра или редактирования, как обновить его в информационной базе.

Показывать занятые файлы при завершении работы

Если установлена и текущий пользователь занял один или несколько файлов, то при завершении работы показывается список этих файлов. Пользователь может открыть файл для просмотра, закончить редактирование, открыть карточку файла.

Показывать колонку "Размер" в списках файлов Если включено, в форме хранилища документов и в списке редактируемых файлов показывается колонка Размер (в Кб).
Показывать информацию, что файл не был изменен При выполнении команды "Закончить редактирование" или "Сохранить изменения", если файл не был изменен, новая версия не создается (существующая не изменяется). При этом выдается диалог с информацией о том, что версия не была создана - если включена настройка Показывать информацию, что файл не был изменен.
Сравнивать версии файлов ODT, DOC, RTF, TXT при помощи

Выбор одного из двух возможных вариантов сравнения файлов указанного формата - "Microsoft Office Word" или "OpenOffice.org Writer".

Настройка открытия текстовых файлов

В колонке Тип файла выбирается вид файлов, которые будут по умолчанию открываться во встроенном текстовом редакторе. Если недостаточно стандартных настроек, внесите свои собственные в колонке Расширение. В колонке Способ открытия можно выбрать, в каком редакторе, внутреннем или внешнем, будут открываться файлы с указанным расширением.

Настройка основного рабочего каталога Нажатие на эту кнопку откроет окно управления рабочим каталогом.
Настройка сканирования Нажатие на эту кнопку откроет окно настройки сканирования.
Настройка распознавания Нажатие на эту кнопку откроет форму настройки распознавания документов и файлов при вводе со сканера.
Настройка ЭП Нажатие на эту кнопку откроет форму настройки электронной подписи.

Работа с документами

Настройка Описание
Вид входящего документа Будет использоваться как вид по умолчанию при создании новых входящих документов.
Способ получения Будет использоваться как способ получения по умолчанию при создании новых входящих документов.
Вид исходящего документа Будет использоваться как вид по умолчанию при создании новых исходящих документов.
Способ отправки Будет использоваться как способ отправки по умолчанию при регистрации исходящих документов.
Вид внутреннего документа Будет использоваться как вид по умолчанию при создании новых внутренних документов.
Показывать предупреждение
при регистрации		 Если флажок установлен, то при регистрации документов будет выдаваться предупреждение о том, что после регистрации поля документов станут недоступны для редактирования.
Организация Указывается организация, к которой относится текущий пользователь. Имеет смысл, если установлена настройка программы "Использовать учет по организациям".
Валюта Указывается основная валюта для документов, в которых присутствует сумма, например, для договоров.
Передача корреспонденту

Указывается поведение программы при передаче документа корреспонденту:

  • Задавать вопрос при передаче корреспонденту
  • Автоматически отражать передачу документа корреспонденту
  • Не отражать передачу документа корреспонденту

Штрихкодирование

Настройка Описание
Показывать форму настройки положения

Если установлена, то при выполнении операций "Вставить штрихкод", "Печать регистрационного штампа" будет открываться окно, в котором пользователю необходимо будет указать положение штрихкода на странице. Значения параметров, указанные пользователем в окне задания настроек положения штрихкода, сохраняются в его персональных настройках.

Если не установлена, то при выполнении указанных операций для определения положения изображения штрихкода будут использоваться значения параметров, определенные в персональных настройках

Положение штрихкода Выбор варианта положения штрихкода. Доступны 4 предопределенных значения, соответствующие углам документа. Пользователь может самостоятельно задать положение изображения штрихкода на странице, выбрав вариант "Произвольное положение"
Отступ слева

Задает смещение изображения штрихкода по горизонтали, считая от левого края страницы.

Если положение штрихкода задано как "произвольное положение", смещение указывается в миллиметрах, считая от левого края страницы.

Отступ сверху

Задает смещение изображения штрихкода по вертикали, считая от верхнего края страницы.

Если положение штрихкода задано как "произвольное положение", смещение указывается в миллиметрах, считая от верхнего края страницы.

Высота штрихкода

Задает высоту изображения штрихкода при его печати или вставке в файл. Если включено отображение цифр в штрихкоде, то высота означает высоту изображения штрихкода вместе с цифрами. При одинаковой высоте штрихкод с цифрами и штрихкод без цифр занимают одинаковое место на листе за счет уменьшения высоты вертикальных штрихов.

Использовать цифры в штрихкоде

Если отметка установлена, то в изображении штрихкода под вертикальными полосками будет размещено значение кода в виде последовательности цифр.

Настроить сканер штрихкодов Позволяет выбрать, какой сканер штрихкодов будет использоваться для поиска документов и файлов по штрихкоду.
Ширина наклейки Ширина наклейки с изображением штрихкода в миллиметрах.
Высота наклейки Высота наклейки с изображением штрихкода в миллиметрах.


Примечание: Настройки штрихкодирования доступны, только если включена настройка программы "Использовать штрихкодирование документов и файлов".

Учет трудозатрат

Настройка Описание
Основной вид работ Указывает вид работ, который будет использоваться по умолчанию при оформлении ежедневных отчетов.
Способ указания времени

Устанавливает один из способов указания длительности работ в ежедневных отчетов:

  • Длительность - для каждой работы указывается ее длительность
  • Время начала - для каждой работы указывается время ее начала, окончание работы совпадает с началом следующей работы
Запрашивать фактические трудозатраты  при выполнении задачи Если установлен, то при выполнении той или иной задачи программа потребует ввести фактические трудозатраты на эту задачу.

Использовать встроенную почту Если установлен, то станет доступным использование почты, встроенной в программу.
Настройки легкой почты Перейти к заданию настроек легкой почты.

Настройка Описание
Проект по умолчанию Указывает проект, который станет подставляться в соответствующий реквизит по умолчанию
Вид проекта по умолчанию Указывает вид проекта, который станет подставляться в соответствующий реквизит по умолчанию

Запрашивать подтверждение  при проверке категорий Если установлен, то при установке отметки о проверке категорий будет запрашиваться подтверждение.
Запрашивать подтверждение при отправке на повторную категоризацию Если установлен, то в результате проверки категорий при отправке на повторную категоризацию будет запрашиваться подтверждение.
Флаг по умолчанию для задач

Указывает цвет для флага, которым по умолчанию помечаются задачи (т. е., все флаги, установленные в списках двойным щелчком мыши, будут такого цвета).
