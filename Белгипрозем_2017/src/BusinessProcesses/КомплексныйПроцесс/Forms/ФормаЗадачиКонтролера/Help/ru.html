В карточке задачи контролера можно следить за ходом и исполнением Комплексного процесса.

Предметом задачи с наименованием "Контролировать" является предмет Комплексного процесса.

Действие, по которому стартовал процесс, отмечается зеленым цветом в списке.

При необходимости контролер может внести изменения в действия, по которым еще не стартовали процессы. В этом случае будет доступна команда Изменить действие.

Если по действию уже стартовал процесс, можно открыть его карточку при помощи одноименной команды.
