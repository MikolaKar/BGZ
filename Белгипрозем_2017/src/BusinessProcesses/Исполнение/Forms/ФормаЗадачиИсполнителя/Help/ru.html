Как задать вопрос по задаче

В процессе работы над задачей вы можете задать вопрос коллеге. Для этого:

  1. Перейдите по гиперссылке Задать вопрос (если по задаче уже задан хотя бы один вопрос, ссылка будет иметь вид - Вопросы по задаче (1)). 
  2. Введите адресата (по умолчанию указан автор задачи) и текст вопроса. 
  3. При необходимости установите Важность вопроса, Срок ответа и прикрепите дополнительные файлы.
  4. Нажмите на кнопку Задать вопрос и закрыть.

Если гиперссылка для создания вопроса недоступна, проверьте, установлена ли настройка программы Использовать рассмотрение вопросов выполнения задач.

Вы можете остановить или прервать рассмотрение отправленного вопроса соответствующими командами в командной панели его карточки.

Как перенести срок выполнения задачи

Если по каким-то причинам вы не успеваете выполнить задачу вовремя, отправьте автору запрос на перенос срока. Для этого:

  1. Перейдите по гиперссылке Срок либо выполните команду Согласовать перенос срока подменю Все действия.
  2. В карточке нового запроса на перенос срока введите желаемую дату и объясните причины переноса. 
  3. При необходимости укажите Важность, Срок ответа и прикрепите дополнительные файлы.
  4. Нажмите на кнопку Задать вопрос и закрыть.

Если ни один из способов создания нового запроса не доступен, то либо автор не ограничил срок выполнения задачи, либо не установлена настройка программы Использовать рассмотрение вопросов выполнения задач.   

Вы можете остановить или прервать отправленный запрос соответствующими командами в командной панели его карточки.

Как отменить выполнение задачи

Если исполнитель ошибочно отметил задачу как выполненную, он может может быстро вернуть ее на исполнение. Для этого в карточке задачи предусмотрена кнопка Отменить выполнение.

Если кнопка Отменить выполнение скрыта, значит процесс перешел на следующий шаг и отменить задачу уже нельзя. В этом случае рекомендуем связаться с автором процесса и договориться об остановке или перезапуске процесса.

Отмена выполнения заносится в Историю событий задач в панели навигации карточки процесса.

Отмена выполнения возможна:

  • Если отмену выполняет исполнитель задачи, его руководитель или делегат;
  • Если выбрано параллельное исполнение (Всем сразу или Смешано);
  • Если процесс не перешел дальше по маршруту.
