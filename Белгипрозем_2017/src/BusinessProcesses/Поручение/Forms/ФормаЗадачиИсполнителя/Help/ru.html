Как принять задачу к исполнению

Нажмите Принять к исполнению в командной панели карточки задачи.

В списке задач принятая задача отображается обычным, а не жирным шрифтом.

Если задача направлена по ролевой адресации, то при выполнении команды Принять к исполнению адресация будет изменена. Задача направится лично сотруднику.

Полезный совет:
Это удобно использовать для "захвата" задач в том случае, если исполнителями роли являются сразу несколько сотрудников.

Отменить принятие к исполнению можно при помощи соответствующей команды в меню Все действия в карточке задачи.

Для завершения задачи служат кнопки в нижней части карточки. Варианты завершения задачи зависят от процесса, в рамках которого она выполняется.

Задача может быть перенаправлена другому исполнителю с использованием персональной или ролевой адресации. Для этого служит команда Перенаправить.

Если в задаче не определен предмет, то исполнитель, при необходимости, может установить его сам с помощью команды Добавить предмет. При этом предмет будет установлен для всех незавершенных задач по текущему процессу, а также в самом процессе.

Как перенести срок выполнения задачи

Если по каким-то причинам вы не успеваете выполнить задачу вовремя, отправьте автору запрос на перенос срока. Для этого:

  1. Перейдите по гиперссылке Срок либо выполните команду Согласовать перенос срока подменю Все действия.
  2. В карточке нового запроса на перенос срока введите желаемую дату и объясните причины переноса.
  3. При необходимости укажите Важность, Срок ответа и прикрепите дополнительные файлы.
  4. Нажмите на кнопку Задать вопрос и закрыть.

Если ни один из способов создания нового запроса на перенос срока не доступен, то либо автор не ограничил срок выполнения задачи, либо не установлена настройка программы Использовать рассмотрение вопросов выполнения задач.

Вы можете остановить или прервать отправленный вопрос соответствующими командами в командной панели его карточки.

Как задать вопрос по задаче

В процессе работы над задачей вы можете задать вопрос коллеге. Для этого:

  1. Перейдите по гиперссылке Задать вопрос (если по задаче уже задан хотя бы один вопрос, ссылка будет иметь вид - Вопросы по задаче (1)).
  2. Введите адресата (по умолчанию указан автор задачи) и текст вопроса.
  3. При необходимости установите Важность вопроса, Срок ответа и прикрепите дополнительные файлы.
  4. Нажмите на кнопку Задать вопрос и закрыть.

Если гиперссылка для создания вопроса недоступна, проверьте, установлена ли настройка программы Использовать рассмотрение вопросов выполнения задач.

Вы можете остановить или прервать рассмотрение отправленного вопроса соответствующими командами в командной панели его карточки.
