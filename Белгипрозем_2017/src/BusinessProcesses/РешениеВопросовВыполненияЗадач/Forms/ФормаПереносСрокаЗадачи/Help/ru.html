Создание нового запроса на перенос сроков

Создать новый запрос на перенос срока можно при помощи команды Создать запрос на перенос срока из карточки задачи (меню Все действия) или из любого списка задач.Откроется карточка создания нового запроса. При создании обязательными для заполнения являются поля:

  • новый срок,
  • обоснование.

Дополнительно можно:

  • прикрепить важный для обоснования запроса файл,
  • посмотреть, не отправлялось ли что-то подобное ранее (вкладка Запросы на перенос срока).

Согласовать перенос срока можно только для задач с установленным ранее сроком. Если при создании задачи автор не установил срок, команда Согласовать перенос срока выдаст ошибку.

В начало

Согласование запроса

Запрос на перенос срока поступает автору как задача с наименованием Согласовать перенос срока по задаче.

В карточке есть все необходимое, чтобы автор мог ознакомиться с ситуацией и принять решение:

  • переписка по согласованию сроков в рамках текущей задачи (поле История вопроса),
  • ссылка на задачу,
  • ссылка на процесс.

Автор может принять одно из двух решений:

  • перенести срок,
  • отказать в переносе.

Дополнительно есть возможность:

  • добавить файл,
  • вписать комментарий.

Бывает, что перенос срока текущей задачи может привести к нарушению срока других задач по процессу, а также самого процесса. В таких случаях в карточке согласования переноса срока появится предупреждение.

При переносе срока процесса Исполнение, необходимо вручную изменить сроки для всех исполнителей в карточке процесса.

В начало

Ознакомление с результатами согласования

Принятое решение поступает автору запроса как задача с наименованием Ознакомиться.

Автор запроса может:

  • закрыть вопрос, если он удовлетворен решением (кнопка Закрыть вопрос)
  • уточнить причину отказа (кнопка Уточнить)

При уточнении причины отказа необходимо ввести текст в поле Комментарий. Если требуется можно добавить файл.

Уточнение снова придет автору как задача с наименованием Согласовать перенос срока по задаче. Таким образом, начнется еще один цикл согласования переноса срока.

В начало

Анализ переноса сроков

Просматривать и анализировать сведения о переносах срока удобно при помощи отчета Переносы сроков задач.

Различные варианты отчета дают возможность узнать:

  • кто из пользователей, и за какой период времени лидирует по количеству запросов на перенос срока;
  • в каком подразделении большего всего запросов, в том числе и за какой-то конкретный период;
  • насколько в среднем по компании и по подразделениям переносятся сроки;
  • какова скорость согласования переноса сроков в подразделениях.

Сформировать отчет Переносы сроков задач можно в разделе Совместная работа при помощи команды Переносы сроков задач панели действий.

Отчет можно дополнительно отфильтровать по:

  • подразделениям,
  • периоду времени.
