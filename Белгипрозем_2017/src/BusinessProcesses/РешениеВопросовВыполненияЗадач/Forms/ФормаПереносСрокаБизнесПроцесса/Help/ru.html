В помощью данного окна вы можете изменить общий срок выполнения процесса вместе со сроком его подзадач. Для этого:

  • Установите желаемый срок выполнения,
  • Впишите причину переноса срока в поле Комментарий,
  • Ознакомьтесь со списком подзадач, срок выполнения которых будет изменен автоматически,
  • Нажмите Перенести срок.

Все переносы срока по процессу созраняются в списке История переносов срока. К списку можно перейти по одноименной ссылке в панели навигации карточки процесса.
