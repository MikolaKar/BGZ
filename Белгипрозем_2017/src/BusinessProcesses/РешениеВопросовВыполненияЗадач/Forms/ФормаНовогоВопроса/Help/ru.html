Кому можно направить вопрос

Как прикрепить к вопросу дополнительные файлы

На что обратить внимание перед созданием нового вопроса 

Перед вами инструмент, с помощью которого удобно обсуждать различные вопросы по задачам. Больше нет необходимости отправлять вопросы по почте, привязывать к ним ссылки на задачи и в дальнейшем искать ответы в переписке.

  • Задайте вопрос по задаче из данного окна,
  • Прикрепите дополнительные файлы,
  • Получите ответ в виде задачи Ознакомление,
  • Все заданные вами вопросы сохранятся на закладке Все вопросы по задаче.

Кому можно направить вопрос

Вопрос по задаче можно направить любому пользователю. По умолчанию в поле Кому указан автор задачи.

Но при выборе адресата следите за тем, чтобы у него были права на работу с задачей.

В начало

Как прикрепить к вопросу файлы

К вопросу можно прикрепить файлы. Это могут быть снимки экрана, видео, документы и т.д.

Для этого:

  1. Сначала введите текст вопроса на закладке Новый вопрос
  2. Затем нажмите  в командной панели списка Файлы и выберите нужный файл.

 

Обратите внимание, что файлы будут прикреплены только к вопросу, а не к задаче, в рамках которой он возник.

В начало

На что обратить внимание перед созданием нового вопроса

  • Перед тем, как создать новый вопрос, рекомендуем проверить, не был ли он задан ранее.

Это можно сделать на закладке Все вопросы по задаче. Список содержит не только вопросы, но и запросы на изменение срока по данной задаче.

  • Если вы ошибочно отправили вопрос, Остановите или Прервите его рассмотрение соответствующими кнопками в командной панели его карточки.

Перейти к карточке вопроса можно дважды кликнув по нему в списке Все вопросы по задаче.
