Отчет вызывается командой панели действий "Ориентировочны объем работ" раздела "Работы, услуги, производство".

Отчет предназначен для показа состояния (местонахождения) дел, зарегистрированных в определенный период, по которым работы находятся в стадии завершения или уже выполнены.

Управление текущими отборами осуществляется с помощью пользовательских настроек (список доступных (быстрых) настроек размещен ниже кнопки "Сформировать"). Полный список настроек доступен при выполнении команды "Настройка...". В настройках для данного отчета можно указать конкретное подразделение.

Для построения отчета необходимо установить использование параметров отбора и указать их значения, нажать кнопку "Сформировать".

Настройка варианта может быть изменена, а также созданы другие варианты отчета.
