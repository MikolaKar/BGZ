Этот отчет позволяет получить информацию по объемам выполненных работ и начисленной заработной плате сводно по подразделениям.

Контрольная сумма в отчете отличная от нуля показывает, что есть расхождение между начисленной ЗП по подразделению и суммами начисленной ЗП по актам по тому же подразделению. Двойным кликом мышкой по Контрольной сумме можно получить ее расшифровку по актам выполненных работ, что бы выяснить причину проблемы.
