Отчет вызывается командой панели действий "Реестр льготируемых работ" раздела "Договоры".

Отчет предназначен для формирования списков реестров выполненных работ по разным применяемым льготам по НДС.

Для построения отчета необходимо выбрать нужный вариант, установить использование параметров отбора и указать значения отбора, нажать кнопку "Сформировать".

Особое внимание нужно обратить на соответствие варианта отчета выбранному параметру Вид льготы!

Настройка каждого варианта может быть изменена, а также созданы другие варианты отчета.
