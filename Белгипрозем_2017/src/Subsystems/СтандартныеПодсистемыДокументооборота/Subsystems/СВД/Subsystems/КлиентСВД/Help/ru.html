Клиент СВД служит для отправки и приема сообщений с помощью сервера СВД.

Для настройки клиентской части СВД нужно выполнить следующие действия:

  1. Установите флажок Использовать СВД в настройках программы на закладке Обмен данными. В разделе Настройка и администрирование появится группа команд СВД.
  2. Создайте новый транспорт СВД (раздел Настройка и администрирование команда СВД - Транспорты)
    • Указать Способ доставки.
    • Указать перечень участников СВД, для обмена документами с которыми будет использоваться этот транспорт. Для каждого участника нужно указать пару реквизитов – Корреспондент и Имя, под которым этот участник зарегистрирован на сервере СВД. Свою организацию также надо включить в список участников.
  3. Создать новое правило отправки сообщений для только что созданного транспорта (раздел Настройка и администрирование команда СВД – Правила отправки сообщений). Правило указывает, документами какого вида надо обмениваться между вашими организациями и корреспондентами через указанный транспорт СВД.
