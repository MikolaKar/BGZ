Справочник "Сборник норм времени"

Справочник содержит информацию о нормах времени на землеустроительные работы, выполняемые по договорам с юридическими и физическими лицами, индивидуальными предпринимателями.

Справочник вызывается для ввода и редактирования информации из раздела "Сметы" / Панель навигации / "Сборники норм времени".

В форме сборника норм времени показывается подчиненная ему информация: таблицы норм времени и примечания сборника.
