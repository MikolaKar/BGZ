Для работы с реквизитами файла (автор, название, описание и т.д.) используется карточка ("Открыть карточку"), а для работы с самим файлом - соответствующее приложение, установленное на компьютере. При редактировании файла его предыдущая версия автоматически сохраняется в информационной базе.

Создание файлов

Для создания нового файла:

  • Откройте список файлов;
  • Выберите нужную папку; 
  • Нажмите клавишу Insert или используйте команду " Создать";
  • Выберите способ создания - "из шаблона", "из файла на диске" или "Со сканера".

Вариант создания файла "Со сканера" становится доступным, если на компьютере подключена компонента для работы со сканером.

Работа с файлами

Для работы с файлами используйте команды, расположенные на командной панели или в подменю "Все действия" в списке файлов или карточке файла: 

Команда Описание
 Редактировать (F4) Помечает файл как занятый для редактирования и открывает его внешним приложением.
 Закончить редактирование Обновляет файл в хранилище и снимает с него пометку занятости для редактирования 
 Сохранить версию Сохраняет версию редактируемого файла в хранилище, но оставляет его занятым текущим пользователем для редактирования
 Занять Помечает файл в хранилище как занятый текущим пользователем для редактирования, но не открывает внешнее приложение. Приступить непосредственно к редактированию можно будет позже.
 Отменить редактирование Снимает с файла пометку занятости для редактирования
 Открыть Открывает файл внешним приложением для просмотра или редактирования, в зависимости от того, помечен ли файл как занятый текущим пользователем для редактирования.
 Сохранить как Сохраняет файл в указанный каталог на жестком или сетевом диске 

 

Если включено использование учета доступа к персональным данным, то в карточке файла появляется список физических лиц, персональные данные которых хранит этот файл. Если владельцем файла не является "Папка файла", а, например, файл присоединен к входящему документу, то этот список недоступен для изменения вручную, а заполняется автоматически из аналогичного списка владельца. Также при этом будет осуществляться протоколирование доступа к полю "Описание" файла в журнал регистрации. Не рекомендуется использовать другие поля файла (например, "Наименование") для хранения персональных данных физических лиц.

Файл может быть отправлен по электронной почте. Для этого в списке файлов или карточке файла следует выполнить команду "Отправить по электронной почте". На экран выводится форма отправки для ввода данных письма (получатель, тема, описание, вложенные файлы).

В списке файлов можно переключить режим просмотра: "списком", "по папкам":

  • Списком - отображается только список файлов

    По папкам - слева на форме отображается структура папок, справа отображается список файлов. В список попадают файлы, хранящиеся в выбранной папке. 

    Создание объектов на основании файлов

    На основании одного или нескольких файлов можно создавать документы, процессы и мероприятия. Для этого необходимо:

    1. Выделить нужные файлы в списке левой кнопкой мыши + Shift;
    2. Выполнить команду Создать на основании из панели действий.

    К новому документу/мероприятию/процессу выбранные вами файлы прикрепятся автоматически.

    Обратите внимание на принцип присоединения файлов:

    • В процесс файлы прикрепляются ссылками. (Изменение исходного файла отразится и в процессе.)
    • В документ файлы копируются. (Изменения исходных файлов не повлияют на их копии в документе.)

     

     

  • Примечание: Если файл зашифрован, полнотекстовый поиск по его содержимому не работает.
