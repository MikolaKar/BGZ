Справочник "Этапы работ" служит для хранения всей информации, связанной с выполнением работ по договору.

Этапы работ подчинены справочнику "Внутренние документы" с видом документа "Договора...". Доступ к справочнику "Этапы договоров" осуществляется из формы договора по команде из панели навигации "Этапы договоров".

Форма этапа работ зависит от того, с кем заключен договор: с физ лицом или юр лицом. Выбор вида работ выполняется из разных справочников видов работ: "Виды работ для физ лиц" и "Виды работ для юр лиц".

В этапе работ по договору с физ лицом отсутствуют некоторые реквизиты, связанные с объектом работ, смета.  

Для этапа работ следует задать:

  • Номер этапа - порядковый номер этапа. К номерам привязана очередность вывода этапов в печатные формы договоров, нумерация актов выполненных работ.
  • Договор - договор, к которому относится этап работ. При создании этапа договора из формы договора реквизит заполняется автоматически.
  • Вид работ - вид работ, выполняемый на данном этапе. В соответствии с выбранным видом работ заполняется цена и рассчитывается стоимость работ по этапу договора.
  • Цена - цена выполнения работ на одном участке. Автоматически подбрасывается при выборе вида работ действующая на дату регистрации договора цена из Прейскуранта. При изменении цены - пересчитывается сумма, НДС и общая стоимость работ.
  • Количество участков - используется для расчета стоимости и для печати в договорах и актах выполненных работ.
  • Стоимость, Стоимость с НДС - рассчитываются автоматически, при изменении вида работ, цены, количества участков, ставки НДС. 
  • Ставка НДС, Сумма НДС, Освобождение от НДС - при выборе ставки НДС "Без НДС" появляется возможность указать ссылку на законодательный акт, который обосновывает применение данной ставки НДС. Ссылка печатается в договоре и в актах выполненных работ.
  • Порядок выполнения совместно с другим этапом - при установленном флажке выбор количества дней выполнения работ по данному виду осуществляется из "параметра вида работ" дни "совместно", иначе "отдельно".
  • Шаблон условий договора - позволяет выбрать из списка условий договора конкретный набор условий.
  • Срочность - признак срочного выполнения работ.
  • Длительность выполнения - позволяет установить произвольную длительность выполнения работ (например, когда установлен флажок Срочность).
  • Источник финансирования - используется для печати в договорах и группировок в отчетах.
  • Объект работ - используется только для договоров юр лиц. Содержит ссылку на объект работ. Заполняется автоматически из предыдущего этапа работ. Используется для печати в договорах и расчете смет.
  • Месторасположение - местонахождение объекта/участка работ. Для физ лиц берется из поручения из реквизита "Адрес земельного участка", для юр лиц из месторасположения Объекта работ. Заполняется автоматически из предыдущего этапа работ.
  • Подразделение - подразделение, которое выполняет работы по данному этапу договора.
  • Смета - используется только для договоров юр лиц. Содержит ссылку на смету работ. 
  • Таблица "Исполнители" - служит для регистрации исполнителей работ, определения размеров начислений заработной платы.

 

Для создания Сметы следует использовать кнопку "Создать смету" на панели инструментов.

ВАЖНО!
При внесении изменений в смету, которая уже выбрана в этапе договора, надо смету перевыбрать в этапе договора еще раз.
