Справочник "Строки таблиц" служит для хранения информации о строках таблиц норм времени, которые используются при расчете смет.

Назначение реквизитов справочника:

Код - внутренний код элемента, который заполняется автоматически при записи элемента;

Владелец - элемент справочника "Таблица норм времени", которому принадлежит строка таблицы;

Номер строки - Порядковый номер строки в таблице. Используется для сортировки строк в таблице, а также для привязки действия примечаний и стадий выполнения работ.

Номер подстроки - Дополнительная нумерация строки в таблице. Применяется в случаях, когда в одной строке таблицы задаются интервалы значений параметра строки и для каждого интервала действуют свои нормы времени. Используется для сортировки строк в таблице, а также для привязки действия примечаний и стадий выполнения работ. Если строка одна, без интервалов, можно не заполнять.

Наименование - Наименование строки. Используется для отображения строк в таблице. Формируется автоматически на основании введенных реквизитов. Может редактироваться вручную.

Использовать условия строки - Установленный флажок открывает видимость реквизитов для ввода информации об условии действия строки: 

Параметр объекта - значение параметра объекта, которое используется в условии.

Нижняя и Верхняя граница диапазона - 2 параметра, которые определяют количественные границы диапазона, в котором действует колонка. Используются совместно с полем "Вариант расчета". Если вариант расчета установлен "Интервал" или "Каждые последующие в интервале" , тогда значение параметра объекта, которое указано в колонке, должно быть больше или равно нижней границы и меньше верхней границы диапазона. Если вариант расчета установлен "Сверх", тогда значение должно быть больше либо равно нижней границе, при этом верхняя граница не учитывается.

Вариант расчета - Определяет как будут использоваться нижняя и верхняя границы. Может принимать следующие значения:

Интервал - значение параметра объекта, которое указано в строке для условия, должно быть больше или равно нижней границы и меньше верхней границы диапазона.

Каждые последующие - значение параметра объекта, которое указано в строке для условия, должно быть больше или равно нижней границы. Для этого варианта становятся доступными и видимыми дополнительные реквизиты "Для последующих" и "Предыдущий элемент".

Сверх- значение параметра объекта, которое указано в строке для условия, должно быть больше либо равно нижней границе, при этом верхняя граница не учитывается.

Каждые последующие в интервале - значение параметра объекта, которое указано в строке для условия, должно быть больше или равно нижней границы и меньше верхней границы диапазона. Для этого варианта становятся доступными и видимыми дополнительные реквизиты "Для последующих", "Коэффициент" и "Предыдущий элемент".

Для последующих - Задает количественный промежуток, на основании которого рассчитывается количество интервалов, которые затем используются в формуле расчета нормы времени.

Коэффициент - Задает коэффициент, который может использоваться в формуле расчета нормы времени.

Предыдущий элемент - Ссылка на предыдущую строку, из которой следует взять базовое значение нормы времени для использования в формуле расчета нормы времени для данной строки.

Использовать количество - Установленный флажок открывает видимость реквизита для ввода информации о параметре объекта, который будет использоваться для расчета стоимости.

Использовать цены - Установленный флажок открывает видимость реквизита для ввода информации о параметре объекта, который будет использоваться для расчета стоимости.

Использовать условия колонок - Установленный флажок означает, что для расчета необходимо использовать условия, задаваемые в колонках таблицы нормы времени.

Параметр объекта для количества или цены - значение параметра объекта, которое используется для расчета стоимости с учетом количественных или ценовых характеристик.
