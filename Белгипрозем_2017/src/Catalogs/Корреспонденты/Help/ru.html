Справочник "Корреспонденты" служит для учета юридических и физических лиц, с которыми Ваша организация обменивается документами. Сведения из этого справочника используются, например, при регистрации входящих и исходящих документов.

Просмотр списка корреспондентов выполняется в разделе "Документы и файлы". В панели навигации следует выполнить команду "Корреспонденты". На экран выводится список корреспондентов. Управление сведениями (добавление нового, копирование, редактирование имеющегося, установка/снятие пометки удаления, поиск) выполняется с помощью команд списка.

В карточке корреспондента сведения о корреспонденте учитываются на двух закладках. На закладке "Основные сведения" учитываются основные описательные данные, на закладке "Адреса, телефоны" - ведется список адресов и телефонов корреспондента. В карточке указывается вид корреспондента - "Юридическое лицо", "Физическое лицо", "Индивидуальный предприниматель", "Юридическое лицо за пределами РФ". Указанный вид корреспондента определяет состав полей карточки, набор дополнительных реквизитов, правила корректного заполнения ИНН, КПП, ОКПО.

Кроме того, в помощью команд панели навигации, размещенных в карточке корреспондента в группе "Перейти", осуществляется управление связанной с корреспондентом информацией: учет контактных лиц, просмотр переписки с контрагентом, внутренние документы, в которых фигурирует корреспондент, просмотр дел (томов), в которых хранятся документы по этому корреспонденту.

При создании корреспондента проверяется его уникальность по реквизитам "ИНН", "КПП", "Наименование" и "Полное наименование". Если в справочнике уже есть записи, у которых значение хотя бы одного из указанных реквизитов совпадает со значением соответствующего реквизита создаваемого корреспондента, и все эти записи доступны текущему пользователю по правам доступа, то откроется список этих объектов. Если среди найденных корреспондентов с дублирующими значениями реквизитов есть хотя бы одна запись, недоступная текущему пользователю по правам доступа, то вместо списка найденных корреспондентов система выдаст сообщение "У пользователя недостаточно прав на исполнение операции над базой данных", и процесс сохранения нового корреспондента прервется.
