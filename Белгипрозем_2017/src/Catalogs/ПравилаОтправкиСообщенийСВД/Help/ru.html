Правила отправки сообщений по СВД регламентируют какие организации, каким корреспондентам, документы каких видов будут отправлять с помощью указанного транспорта.

Дополнительно Вы можете настроить ограничения при отправке сообщений, например, чтобы максимальный размер всех передаваемых файлов не превышал 50 Мб.
