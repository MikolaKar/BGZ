Справочник "Колонки таблиц" служит для хранения информации колонок таблиц норм времени, которые используются при расчете смет.

Назначение реквизитов справочника:

Код - внутренний код элемента, который заполняется автоматически при записи элемента;

Владелец - элемент справочника "Таблица норм времени", которому принадлежит колонка;

Номер колонки - Порядковый номер колонки в таблице. Используется для сортировки колонок в таблице.

Наименование - Наименование колонки. Используется для отображения колонок в таблице. Формируется автоматически на основании введенных ниже реквизитов. Может редактироваться вручную.

Группа "Условия" описывает условия, которые распространяются на действие данной колонки.

Нижняя и Верхняя граница диапазона - 2 параметра, которые определяют количественные границы диапазона, в котором действует колонка. Используются совместно с полем "Вариант расчета". Если вариант расчета установлен "Интервал" или "Каждые последующие в интервале" , тогда значение параметра объекта, которое указано в колонке, должно быть больше или равно нижней границы и меньше верхней границы диапазона. Если вариант расчета установлен "Сверх", тогда значение должно быть больше либо равно нижней границе, при этом верхняя граница не учитывается.

Вариант расчета - Определяет как будут использоваться нижняя и верхняя границы. Может принимать следующие значения:

Интервал - значение параметра объекта, которое указано в колонке, должно быть больше или равно нижней границы и меньше верхней границы диапазона.

Каждые последующие - значение параметра объекта, которое указано в колонке, должно быть больше или равно нижней границы. Для этого варианта становятся доступными и видимыми дополнительные реквизиты "Для последующих" и "Предыдущий элемент".

Сверх- значение параметра объекта, которое указано в колонке, должно быть больше либо равно нижней границе, при этом верхняя граница не учитывается.

Каждые последующие в интервале - значение параметра объекта, которое указано в колонке, должно быть больше или равно нижней границы и меньше верхней границы диапазона. Для этого варианта становятся доступными и видимыми дополнительные реквизиты "Для последующих" и "Предыдущий элемент".

Для последующих - Задает количественный промежуток, на основании которого рассчитывается количество интервалов, которые затем используются в формуле расчета нормы времени.

Предыдущий элемент - Ссылка на предыдущую колонку, из которой следует взять базовое значение нормы времени для использования в формуле расчета нормы времени.
