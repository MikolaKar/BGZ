Предназначен для настройки наборов дополнительных реквизитов (сведений)

Администратор программы может настроить наборы дополнительных свойств для определенных объектов программы. Способ хранения дополнительного свойства в программе и место размещения в окне объекта программы зависит от типа свойства: дополнительный реквизит или сведение.

Предопределенные наборы соответствуют объектам программы (спискам или документам), в которых можно использовать дополнительные реквизиты и сведения. Добавление новых наборов в этом списке невозможно.

Программа позволяет создать набор дополнительных реквизитов (сведений) как для всех элементов списка, так и для части. В этом случае список программы является группой, а элементами группы являются реквизиты списка или значения реквизитов списка (в этом случае количество элементов группы будет равно количеству значений реквизита). Наборы автоматически создаются при создании групп в соответствующих списках. 

Какие сведения выводятся в списке

Список Дополнительные реквизиты (сведения) представляет собой таблицу из двух частей:

  • в левой части располагаются Наборы дополнительных реквизитов (сведений), назначенных справочнику или документу, которые показаны в виде групп. Если в списке имеются группы, то Наборы дополнительных реквизитов (сведений) соответствуют группам справочника.
  • В правой части списка в виде таблицы располагается список дополнительных реквизитов (сведений), входящих в выделенный набор. Жирным шрифтом в списке дополнительных реквизитов (сведений) отражаются реквизиты (сведения), обязательные для заполнения. В таблице выводятся сведения: 
    • Дополнительный реквизит (сведение) - включает перечень наименований дополнительных реквизитов (сведений);
    • Тип значения - указывается тип дополнительных реквизитов (сведений) и количество значений.

Как назначить новый дополнительный реквизит (сведение) списку или документу

В зависимости от настройки может быть два способа ввода новых реквизитов (сведений):

  • Выберите соответствующий этому списку или документу набор, нажмите Создать. По умолчанию дополнительный реквизит (сведение) может быть добавлен только в один набор.
  • Выберите соответствующий этому списку или документу набор, нажмите Добавить - Новый. (Аналогична команде Создать) С помощью этой команды дополнительный реквизит (сведение) может быть добавлен только в один набор.

Общие наборы дополнительных реквизитов (сведений)

Для некоторых списков программы (например, Демо: Номенклатура и Демо: Партнеры) предусмотрены предопределенные наборы Общие. Введенные в такой набор дополнительные реквизиты и сведения по умолчанию становятся общими для всех элементов этих списков.

  • Выберите набор Общий.
  • Нажмите Создать или Добавить - Новый.

Как добавить дополнительные реквизиты (сведения) по образцу

можно использовать ранее введенные дополнительные реквизиты и сведения из других наборов. Можно добавлять реквизиты с общим списком значений сразу для нескольких однотипных реквизитов (например, Цвет двери по образцу реквизита Цвет). Список значений в этом случае используется из реквизита-образца. Удобно выполнять централизованную настройку списка значений сразу для нескольких однотипных реквизитов.

  • Выберите набор из списка.
  • Нажмите Добавить - По образцу.
  • выберите дополнительный реквизит (сведение) из списка уже существующих в программе. 

Как добавить общие дополнительные реквизиты (сведения)

Общий реквизит входит в несколько наборов. Этот вариант подходит для тех случаев, когда общим должен быть не только список значений, но и сам реквизит. В списках и отчетах по элементам различных списков и документов общие реквизиты группируются и выводятся как один реквизит в одной колонке или поле. Например, с помощью общего реквизита Цвет можно отобрать всю желтую одежду и обувь.

  • Выберите набор из списка.
  • Нажмите Добавить - Общий(Общее)
  • Выберите из списка уже существующий общий дополнительный реквизит(сведение). Если в списке нет нужного реквизита, нажмите Создать.

    Особенности удаления дополнительных реквизитов (сведений):

    При пометке на удаление свойства в наборе, значения свойства сохраняется в объектах до тех пор, пока не будут очищены. При этом свойство, помеченное на удаление в наборе, будет отображаться в окне объекта или окне дополнительных сведений особым образом.
    При необходимости можно снять пометку удаления.
