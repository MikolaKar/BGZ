Справочник "Карточки объектов работ" служит для ввода и хранения значений характеристик объектов (параметров), которые будут использованы для расчета смет. 

Карточка вводится на основании входящего документа Поручение (Заявка на выполнение работ)

  • Пакет документов - ссылка на пакет документов по договору и даты предоставления документов.
  • Дата предоставления - дата предоставления пакета документов, заполняется автоматически при проставлении дат в пакетах документов.  

    На закладке "Исходные данные" заполняется общая информация о Заказчике, Объекте работ, Карточке. Частично информация заполняется автоматически из входящего документа-основания.
