Форма проверки выражения предназначена для проверки результата исполнения выражения, написанного на встроенном языке 1С в качестве условия маршрутизации.

Список объектов, на которых можно проверить выражение, определяется типом объектов, для которых создается условие маршрутизации.

Для того, чтобы проверить выражение, необходимо выбрать элемент в списке объектов и выполнить команду "Проверить выражение на выбранном элементе". В зависимости от результата проверки на форме отображается следующая информация:

  • Выражение выполнено, переменной Результат присвоено значение Истина

    В поле "Результат проверки" отображается надпись "Истина". Означает, что условие выполнено.

  • Выражение выполнено, переменной Результат присвоено значение Ложь

    В поле "Результат проверки" отображается надпись "Ложь". Означает, что условие не выполнено.

  • Выражение выполнено, переменной Результат не присвоено значение, либо присвоено значение не типа "Булево"
    В поле "Результат проверки" отображается сообщение об ошибке.
  • При выполнении выражения возникла ошибка

    В поле "Результат проверки" отображается описание причины ошибки, возникшей при выполнении выражения.

    Текст выражения можно отредактировать. При закрытии формы командой "Закрыть" измененный текст выражения будет отображаться на форме задания условия маршрутизации, откуда вызывалась форма проверки выражения.

    Примечание: для того, чтобы при выделении элемента в списке объектов автоматически происходила проверка выражения на выделенном элементе, необходимо установить отметку в галочке "Сразу проверять при выборе документа".
