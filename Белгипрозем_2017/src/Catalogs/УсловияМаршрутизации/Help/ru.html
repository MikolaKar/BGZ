Условия маршрутизации позволяют настроить условное выполнение процессов. Условия маршрутизации используются при работе с шаблонами процессов и определяют необходимость включения того или иного участника в процесс либо о создании процесса из шаблона по документу или файлу.

Условие маршрутизации может быть задано в одном из трех видов:

  • в режиме конструктора 

    Условие задается в виде отбора по реквизитам Предмета процесса. Тип Предмета определяется типов объектов, для которых создается условие маршрутизации.

  • на встроенном языке

    В выражении на встроенном языке обязательно должно быть присвоено значение переменной Результат типа Булево. Если значение переменной Результат не присвоено, то считается, что условие невыполнимо. В выражении можно оперировать переменной Предмет, тип которой определяется типом объетов, для которых создается условие маршрутизации.

    Например, для того, чтобы проверить наличие регистрационного номер у документа, необходимо написать следующее выражение:
     Если НЕ ТипЗнч(Предмет) = Тип("СправочникСсылка.Файлы")
          И ЗначениеЗаполнено(Предмет.РегистрационныйНомер) Тогда
          Результат = Истина;
     Иначе
          Результат = Ложь;
     КонецЕсли;

    Для того, чтобы проверить, есть ли у документа файлы, необходимо написать следующее выражение:
     Запрос = Новый Запрос;
     Запрос.Текст =
          "ВЫБРАТЬ
          |    Файлы.Ссылка КАК Ссылка
          |ИЗ
          |    Справочник.Файлы КАК Файлы
          |ГДЕ
          |    Файлы.ВладелецФайла = &Владелец";
     Запрос.УстановитьПараметр("Владелец", Предмет.Ссылка);
     Результат = Запрос.Выполнить();
     Выборка = Результат.Выбрать();
     Результат = Выборка.Количество() > 0;

    Примечание:     Выражение на встроенном языке выполняется в безопасном режиме.

  • в виде комбинации других условий

     Условие задается как логическое выражение, аргументами которого являются результаты проверки других условий.

    Для создания и редактирования условий маршрутизации в разделе "Нормативно-справочная информация" в панели навигации следует выполнить команду "Условия маршрутизации".
