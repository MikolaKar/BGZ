Справочник "Акты проверок" предназначен для хранения информации по проверкам дел.

Создавать Акт проверки следует из карточки Дела, используя кнопку "Создать Акт проверки". В этом случае практически все реквизиты акта будут заполнены автоматически.

Если у проверяющего есть замечания он должен добавить их в нижней таблице Акта проверки.

Заполнение информации о проверяющем, времени проверки, оценке происходит в автоматическом режиме в момент выполнения действий с задачами по процессам "Проверка дела"

С помощью кнопки "Акт проверки" можно получить печатную форму Акта проверки.
