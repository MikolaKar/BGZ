В списке на закладке "Темы обсуждений" представлены все темы, которые обсуждаются в Вашей организации в текущий момент. Темы можно сгруппировать по разделам. Для быстрого поиска нужных тем, Вы можете предварительно отсортировать их список. Например, выбрать просмотр только прочтенных или не прочтенных тем, а также только обсуждений по документам. Также Вы можете воспользоваться полнотекстовым поиском в данных.

В списке на закладке "Тема: Название темы" представлены сообщения темы обсуждения. Он формируется в хронологической последовательности, каждому сообщению присваивается свой порядковый номер. В списке Вы можете узнать, были ли прикреплены дополнительные файлы к сообщению, начато ли по нему голосование, в ответ на какое сообщение оно было написано. При помощи соответствующих команд Вы можете фильтровать сообщения, например, просматривать ответы только на конкретное сообщение.

Как создать новую тему и написать в ней сообщение

Вы можете создать сообщение обсуждения только в рамках какой-либо темы, существующей или новой.

Чтобы создать новую тему обсуждений, выполните следующие действия:

  1. Нажмите на кнопку Создать командной панели, в новом окне откроется карточка темы.
  2. Введите название темы (поле Название), а также краткое описание темы.
  3. При необходимости выберите или создайте раздел (поле Раздел), в который она будет входить.
  4. Нажмите на кнопку Записать и закрыть, чтобы завершить создание темы.

Чтобы создать новое сообщение обсуждения, выполните следующие действия:

  1. Выберите тему из списка, в на закладке темы откроется список сообщений этой темы.
  2. Нажмите на кнопку Написать, в новом окне откроется карточка нового сообщения.
  3. Введите текст сообщения, при необходимости прикрепите файл, который поясняет суть сообщения.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить сообщение.

Удобнее всего начинать обсуждение из карточек данных программы, например внутренних документов или мероприятий, в этом случае программа автоматически создаст новую тему, заполнит предмет темы и ее сообщений.

Как создать новый раздел (группу тем)

Для быстрого поиска тем обсуждений группируйте их по разделам. Чтобы создать новый раздел:

  1. Установите режим просмотра списка По разделам.
  2. Выполните команду , в новом окне откроется карточка раздела.
  3. Введите название раздела в поле Наименование.
  4. Нажмите на кнопку Записать и закрыть, чтобы завершить создание раздела.

Вы можете переместить тему в раздел путем перетаскивания мышкой в режиме просмoтра списка По разделам.

Как запретить добавление сообщений по теме

Если Вы хотите запретить дальнейшее добавление сообщений к теме обсуждения, автором которой являетесь, закройте тему.

Чтобы закрыть тему обсуждения:

  1. Выберите тему из списка.
  2. Выполните команду  (клавиша F2), в новом окне откроется карточка темы.
  3. Установите флажок Закрытая.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.

Как настроить область чтения

Чтобы настроить расположение области чтения (справа, снизу, отключить), а также шрифт, перейдите в меню Все действия и выполните команду Область чтения.

Как начать процесс по теме обсуждения

Для организации выполнения работ по теме обсуждения используйте процессы.

Чтобы создать новый процесс по теме обсуждений:

  1. Выберите тему из списка.
  2. Выполните команду Процесс подменю Создать на основании, в новом окне откроется помощник создания процессов.
  3. Выберите нужный процесс из списка, заполните все необходимые для старта поля в его карточке.
  4. Нажмите на кнопку Стартовать и закрыть, чтобы начать процесс по теме.

Чтобы просмотреть все процессы и задачи по теме:

  1. Выберите тему из списка, в новом окне откроется список сообщений темы.
  2. Перейдите по гиперссылке Тема, откроется карточка темы обсуждения
  3. Выполните команду Открыть процессы и задачи... подменю Все действия, в новом окне откроется список процессов и задач.

Как посмотреть ответы на сообщение

Вы можете сформировать список сообщений таким образом, чтобы просматривать только ответы на какое-то конкретное сообщение. Для этого:

  1. Выберите сообщение из списка.
  2. Выполните команду Показать ответы на сообщение в подменю Все действия, будет сформирован список ответов на выбранное сообщение.

Чтобы вернуться к просмотру всего списка сообщений, еще раз выполните команду Показать ответы на сообщение в подменю Все действия.

Вы также можете настроить, чтобы по умолчанию список сообщений открывался в виде дерева. Для этого выполните команду Изменить вид отображения сообщений в подменю Все действия.

Как начать процесс по сообщению обсуждения

Для организации выполнения работ по сообщению обсуждения Вы можете воспользоваться процессами.

Чтобы стартовать процесс по сообщению обсуждения:

  1. Выполните команду Процесс в подменю Создать на основании, откроется помощник создания процессов.
  2. Выберите нужный процесс из списка.
  3. Заполните карточку процесса.
  4. Нажмите на кнопку Стартовать и закрыть, чтобы начать процесс.

Чтобы посмотреть все процессы и задачи по сообщению:

  1. Откройте карточку нужного сообщения из списка.
  2. Выполните команду Открыть процессы и задачи... , в новом окне откроется список всех процессов и задач по сообщению.

Как начать голосование по сообщению

Для сбора и анализа мнений коллег по вопросу, который вы поднимаете в сообщении обсуждения, начните по нему голосование:

  1. Откройте карточку сообщения обсуждения.
  2. Установите флажок Голосование, станет доступна закладка Настройка голосования.
  3. Введите вопрос, по которому будет проводиться голосование (поле Вопрос голосования).
  4. При необходимости отредактируйте текст вариантов голосования.
  5. Нажмите на кнопку Записать и закрыть, чтобы начать голосование.

После начала голосования, напротив сообщения будет автоматически установлен значок . Посмотреть количество проголосовавших и процентное соотношение между ответами Вы можете в карточке соответствующего сообщения.
