Справочник "Примечания норм времени"

Справочник содержит информацию о примечаниях к сборнику норм времени, в которых содержится информация уточняющая или корректирующая применение норм времени.

Справочник вызывается для ввода и редактирования информации из раздела "Сметы" / Панель навигации / "Сборники норм времени".
