Проект - это временное предприятие, предназначенное для создания уникальных продуктов, услуг или результатов. Используется для автоматизации исполнения плана проекта, а также для ведения аналитического учета всех данных программы (документов, файлов, мероприятий, писем, процессов и т.д.).

В результате в карточке проекта вы можете посмотреть:

  • список всех файлов, документов, мероприятий, которые относятся к проекту;
  • переписку по проекту;
  • трудозатраты на работу по проекту или его задачам и многое другое.

Для исполнения проектных задач используйте процессы. Процессы по проектам разграничены на Основные, направленные на выполнение проектных задач и Вспомогательные, которые не влияют на результат выполнения проектной задачи.

Текущее состояние проектных задач, соответствие фактических трудозатрат плановым вы можете узнать при помощи разных вариантов плана проекта.  Контролировать состояние проектных задач удобнее всего по диаграмме Ганта. Контролировать проектные задачи и анализировать трудозатраты вы можете при помощи отчетов Проекты и Проектные задачи.

Создавать проекты могут только пользователи, которые входят в группу доступа Руководители проектов.

Вы также можете загружать данные о проектах из Microsoft Project и организовывать их исполнение в программе. В качестве источника данных используйте файл проекта, выгруженный в XML.
