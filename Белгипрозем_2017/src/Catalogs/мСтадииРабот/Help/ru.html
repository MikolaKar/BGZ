Справочник "Стадии работ" служит для хранения стадий работ, которые будут использованы при расчете смет.

Для стадии работ следует задать:

  • Полное наименование - текст, который будет выводиться при печати сметы в графе "Стадии работ".
  • Приоритет - регулирует очередность расположения стадий работ в смете. Стадии выводятся в порядке возрастания приоритета.
  • Не входит в объем для начисления ЗП - Сумма по стадии работ будет исключаться из объема работ для начисления заработной платы при актировании.
