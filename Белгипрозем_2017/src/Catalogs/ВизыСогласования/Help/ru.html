Визы - это результаты согласования документов. Они накапливаются и хранятся в карточке внутреннего, входящего или исходящего документа.

Как отметить результат за другое согласующее лицо

Автор визы может отметить результат за согласующее лицо.

Чтобы отметить результат за другое согласующее лицо, выполните следующие действия:

  1. Выберите согласующее лицо из списка или введите с клавиатуры вручную
  2. Нажмите на кнопку Отметить результат за согласующее лицо
  3. Выберите результат согласования из списка или введите с клавиатуры вручную
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить визу

После выбора результата в поле Установил записывается лицо, которое проставило визу.

Как подписать визу электронной подписью

При необходимости можно подписать визу электронной подписью. Подписывается только сам факт и результат согласования документа. В результате виза становится недоступна для изменения.

Чтобы подписать визу электронной подписью, выполните следующие действия:

  1. Выполните команду Подписать контекстного меню ЭП
  2. В открывшемся окне выберите сертификат электронной подписи
  3. Введите пароль к закрытому ключу в поле Пароль
  4. При необходимости введите комментарий к подписи
  5. Нажмите на кнопку Подписать, чтобы подписать визу электронной подписью

Настроить подписание визы электронной подписью можно также из карточки процесса Согласование. Для этого при создании процесса нужно установить флажок Подписывать ЭП при согласовании.

Общие сведения о визах согласования

Визы согласования накапливаются и хранятся во вкладке Визы карточки документа. Ознакомиться с историей ранних циклов согласования (если они были) можно также из вкладки Визы при помощи команды История согласования.
