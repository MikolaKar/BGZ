Справочник "Объекты работ" содержит информацию по объектам выполняемых работ. Подчиняется справочнику "Корреспонденты".

Используется для печати в договорах, а также для расчета смет.

Справочник двухуровневый. В качестве первого уровня могут выступать только элементы. Второй уровень используется только при расчете смет по договорам с Госкомимуществом.

Поле Масштаб заполняется только для объектов работ по договорам с Госкомимуществом.
