Справочник содержит исполнителей работ по Заявке (входящему документу).

Если по Заявке еще не создан договор, то можно добавлять и удалять исполнителей.

Если по Заявке уже есть договор, то при создании новых этапов договоров, исполнители в этапы будут добавлены автоматически.

Если по Заявке уже есть договор и этапы, которые еще не заактированны, то исполнители работы в таких этапах будут изменены.

При актировании списки исполнителей работ обновляются в этапах договоров и в Заявках.

Если нет незаактированных этапов работ, то изменить список исполнителей в Заявке нельзя.
