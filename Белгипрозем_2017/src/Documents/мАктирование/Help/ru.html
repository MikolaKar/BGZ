Этот документ предназначен для регистрации Актов выполненных работ и расчета сдельной заработной платы.

Назначение таблиц

Первая таблица служит для добавления (выбора) договора и отображения общей информации по договору. Для ввода новой строки следует нажать кнопку Добавить и ввести номер договора. При этом во всех остальных таблицах будет показываться связанная с выбранным договором информация.

Две таблицы в средней части экрана отображают информацию об этапах договора и расчетах по договору. При смене строки в правой и левой нижней таблицах будет показана информация, которая относится к выбранному этапу. Добавлять или удалять строки в этих таблицах нельзя, они носят только информационный характер и служат для выбора этапа работ по договору.

Две таблицы в нижней части экрана предназначены для ввода информации. В левой вводится информация из Акта выполненных работ, в правой - расчет сдельной заработной платы для исполнителей работ по акту.

Таблица актирование работ

Если по этапу работ уже был ранее проведен акт выполненных работ, то он будет показан в таблице. Для ввода новой строки следует нажать кнопку Добавить в командной панели этой таблицы и заполнить поля: 

  • Число - номер дня в месяце, дата акта заполнится автоматически.

  • Сумма - стоимость выполненных работ с НДС.

  • НДС - сумма НДС, если есть.

  • Сумма прочих - стоимость межевых знаков и др., не относящееся к распределяемому объему работ. 

В результате заполнения данных по акту рассчитывается  Объем работ к распределению и Не распределено.

Таблица начисления заработной платы

Таблица служит для распределения объемов работ между исполнителями и начисления сдельной заработной платы.

Назначение полей над таблицей:

  • Объем работ к распределению - Объем работ, подлежащий распределению. Из общей стоимости акта вычитаются НДС и сумма прочих. 

  • Не распределено - нераспределенный объем работ. Служит для контроля правильности распределения. 

  • Остаток - при установленном флажке остаток нераспределенного объема работ будет переноситься на следующего добавляемого сотрудника. 

  • Делить на - если количество больше 0, то объем будет разделен на это количество исполнителей и подставлен во всех строках. Используется для более точного деления объема работ между исполнителями. 

    Назначение полей таблицы:

  • Без названия - флажок, которым отмечаются сотрудники, относящиеся к другим подразделениям. При установленном флажке сумма, начисленная в данной строке, исключается из объема работ к распределению. 

  • Таб. №, Исполнитель, Подразделение - при вводе Таб.№ исполнитель и его подразделение заполняются автоматически, можно выбрать исполнителя из списка пользователей.

  • КТУ - коэффициент трудового участия. По умолчанию равен 1. Может редактироваться вручную, либо автоматически, если установлено количество Делить на.

  • Норматив - процент начисления сдельной заработной платы от объема работ, который берется из Нормативы ЗП.

  • Коэффициент - дополнительный коэффициент для коррекции начисления сдельной заработной платы.

  • Обоснование коэффициента - ссылка на внутренний документ, на основании которого применяется коэффициент.

  • Объем работ - стоимость объема работ, от которого считается заработная плата.

  • Начислено - сумма начисленной сдельной заработной платы.

Список исполнителей, внесенных этим документом, является окончательным и может быть изменен только документом Актирование. Исполнители записываются также в этап договора.
