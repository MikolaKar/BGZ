Документ Взаимозачет предназначен для изменения состояния расчетов с покупателями (заказчиками).

В зависимости от выбранного значения в поле Вид взаимозачета позволяет сделать взаимозачет:

  • Между корреспондентами - между различными корреспондентами
  • Между договорами - между договорами одного и того же корреспондента
  • Между этапами договоров - между этапами как одного так и разных договоров.
