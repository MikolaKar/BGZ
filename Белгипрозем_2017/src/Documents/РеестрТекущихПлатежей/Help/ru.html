Этот документ предназначен для регистрации движения денежных средств по расчетному счету.

В шапке документа указывается дата ввода документа и направление платежа.

Движение денежных средств записывается на Дату выписки банка. Сумма платежей - используется для контроля правильности разноски оплат.

Работа с документом

Ручной ввод платежей.

Для ввода платежа надо нажать кнопку Добавить.

Поля Номер док. и Дата док. служат для ввода номера и даты платежного документа. Затем надо ввести Номер договора. Если договоров будет найдено несколько, то появится список для выбора конкретного договора. После определения договора в оставшихся полях этой таблицы будет показана дополнительная информация по договору, а нижняя таблица заполнится информацией по этапам работ выбранного договора. После ввода Сумма платежа она будет распределена между этапами работ соответственно их сметной стоимости и с учетом оплаты ранее, если такая была. Оплату этапов можно изменить в поле Сумма оплаты этапа договора.  Также необходимо заполнить поле Способ платежа - которое будет потом использовано в Отчете для статистики 4-УН.  

Загрузка платежей из ЕРИП.

Для загрузки платежей из ЕРИП надо нажать кнопку Загрузить из ЕРИП.

В открывшемся окне данной обработки надо указать файл, который будет загружен и нажать кнопку "Загрузить". Данные из файла будут добавлены в табличные части документа. При загрузке происходит автоматический поиск договора по его номеру, проверяется совпадение ФИО заказчика в договоре и в файле. При несовпадении ФИО или не нахождении договора поле Договор остается пустым и выдается сообщение о том, что есть  ошибка в при загрузке строки файла с указанием номера договора и ФИО.

Файл не будет загружен, если дата документа Реестр текущих платежей и дата поступления денежных средств на расчетный счет предприятия, указанная в загружаемом файле будут различаться.

После ввода всех платежей надо нажать кнопку Провести и закрыть.

Отчет Состояние расчетов

Для оперативного получения информации о расчетах с контрагентами можно воспользоваться кнопкой Состояние расчетов и сформировать отчет по движению средств по выбранному контрагенту.
