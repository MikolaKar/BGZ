Документ используется для ввода информации о планируемой работе. Формируется автоматически при записи этапа договора. Регистрирует сметную стоимость этапа договора.

Если вводится дополнительное соглашение, то в основание записывается документ дополнительное соглашение и его договор, а основная часть содержит информацию по основному договору.
