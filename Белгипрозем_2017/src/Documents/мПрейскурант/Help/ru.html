Этот документ предназначен для ввода информации о действующих на предприятии ценах на выполнение работ для физических лиц.

Назначение реквизитов

Эти реквизизты будут использоваться для печати в договорах в качестве обоснования цен:

  • Номер и дата приказа - Номер и дата приказа, которым вводятся новые цены,
  • Должность утвердившего - Кто подписал приказ.

Действие Прейскуранта начинается с Начало действия и действует до тех пор, пока не будет введен новый прейскурант.

Если на предприятии есть определенные категории физических лиц, для которых действуют специальные цены, то для ввода цен для таких категорий лиц необходимо заполнить поле Категория физ лиц.

Если таких категорий физических лиц нет или прейскурант вводится для физ лиц, которые не относятся ни к каким категориям, то заполнять данный реквизит не надо.
