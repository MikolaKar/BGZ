Как работать со списком задач

Помимо списка задач можно посмотреть краткое описание задачи, файлы и переписку по задаче. Обратите внимание на то, что важные задачи выделены курсивом.

Для согласований, утверждений и рассмотрений предусмотрено быстрое визирование в правой нижней части формы.
Двойной щелчок на задаче или письме из закладки Переписка открывает карточку задачи или письма. Двойной щелчок на файле открывает его для просмотра.

В описании задачи  могут присутствовать гиперссылки, с помощью которых можно, например, перейти к просмотру документа-основания и т.п.

В начало

Как найти задачу по автору или наименованию

Если в списке  показано большое количество задач, вы можете найти нужную задачу, для которой известен автор или часть наименования.

Для поиска задач выполните следующие действия:

  • Введите часть фамилии автора задачи или часть наименования в поле Найти.
  • Нажмите .
  • Список задач будет отфильтрован в соответствие с заданными условиями.

    • Для возврата к просмотру полного перечня задач нажмите .

    Как добавить визу

    В правой нижней части окна для наиболее частых задач руководителя предусмотрено быстрое добавление визы:

  • Для задачи рассмотрения введите резолюцию и нажмите Рассмотрено.
  • Для задачи утверждения введите при необходимости комментарий и нажмите Да или Нет, задавая таким образом результат утверждения.
  • Для задачи согласования введите при необходимости комментарий и нажмите Да или Нет, задавая таким образом результат согласования.

    • Для остальных типов задач щелкните на ссылке Перейти в форму для выполнения задачи.
