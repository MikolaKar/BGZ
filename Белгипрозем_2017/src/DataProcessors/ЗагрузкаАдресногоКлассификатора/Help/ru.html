Обработка позволяет заменить данные в адресном классификаторе на новые данные из файлов.

Для загрузки необходимо 2 файла:

  • файл населенных пунктов с кодами СОАТО
  • файл с перечнем улиц населенных пуктов

 

Формат файлов должен быть *.xlsx.
