Обработка предназначена для изменения данных по заработной плате в документах Актирования.

Для работы необходимо указать период, в котором будут внесены изменения в документы. Затем надо указать номер договора в поле "Поиск по номеру договора". Если ни одного договора с таким номером не будет найдено, то в таблицах будут показаны все ранее выбранные договора и расчеты зарплаты по ним из этой обработки.

Кнопка  предназначена для обновления информации по выделенному договору в верхней таблице из базы данных. 

После окончания редактирования данных следует нажать кнопку "Провести" и закрыть обработку.
