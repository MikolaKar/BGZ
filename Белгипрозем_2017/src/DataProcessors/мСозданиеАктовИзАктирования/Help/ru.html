Обработка позволяет создать внутренние документы Акт выполненных работ на основании проведенного документа Актирование.

В поле Актирование надо выбрать документ, из которого будет браться информация для формирования Актов выполненных работ.

В поле Подготовил укажите сотрудника, который будет подписывать Акты выполненных работ.

По кнопке Заполнить из Актирования будет перенесена информация в табличную часть. Флажками можно отметить строки, по которым надо создать АКты выполненных работ.

По кнопке Создать акты будут созданы внутренние документы Акт выполненных работ. Состояние будет автоматически установлено Подготовлен.

По кнопке Установить статус можно изменить статус Актов выполненных работ, отмеченных в табличной части.
