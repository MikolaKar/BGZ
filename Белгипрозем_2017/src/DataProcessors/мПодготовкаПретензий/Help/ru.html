Обработка предназначена для создания исходящих писем с видом Претензия с прикрепленными к ним вордовских документов самих писем.

Порядок работы

 Для начала надо заполнить поля в форме.

Для отбора договоров:

  • Организация - (обязательно) организация, по которой будут отбираться договоры.
  • Вид документа - можно указать определенный вид договора, по которому будут отобраны договора. Если не задан, то по всем.
  • Контрагент - если указан, то будут выбраны данные только поэтому контрагенту, иначе по всем.
  • Дата остатков - Дата, на которую будут выбраны остатки задолженностей. Письма создаются текущей датой.
  • Дней просрочки - количество дней, прошедших после подписания актов.

 

Для записи в исходящих документах:

  • Номенклатура дел - номенклатура дел, к которой относятся письма с претензиями. 
  • Вопрос деятельности - вопрос деятельности, к которому относятся письма с претензиями.
  • Подписал - ответственное лицо, кто будет подписывать письма-претензии.
  • Подготовил - ответственное лицо, создающее письма-претензии.
  • Тема - тема письма, которая будет подставлена во всех письмах.

Верхняя таблица содержит перечень корреспондентов, которым можно направить письма.

Нижняя таблица содержит перечень договоров и этапов, по которым есть задолженности.

Работа по подготовке претензий выполняется в 3 шага:

  1. Нажать Заполнить - в результате таблицы будут заполнены договорами, по которорым есть просроченная задолженность. Верхняя таблица заполняется итогами по корреспондентам. Если в течение этого дня уже формировались претензии, то они будут показаны в верхней таблице.
  2. Отметить флажками строки в верхней таблице. Нажать Создать исходящие письма - в результате по отмеченным корреспондентам будут созданы искходящие документы с прикрепленными к ним вордовскими файлами. При повторном нажатии Создать исходящие письма новые исходящие документы не создаются, в уже созданных удаляются вордовские документы и взамен создаются новые.
  3. Печать - выводит отмеченные письма в вордовский документ для просмотра и печати. Если необходимо редактирование созданных файлов, то это недо сделать до нажатия Печать. Надо в нужной строке открыть Карточку файла, отредактировать и записать файл.   

При выборе строки в верхней таблице с корреспондентом в нижней автоматически показываются договоры только этого корреспондента. По кнопке Показать все можно просмотреть все договоры.
