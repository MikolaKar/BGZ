Создает и проводит документы Актирование, подбирая Акты выполненных работ.

Для создания документов необходимо указать:

Дата актирования - дата, на которую будут создаваться документы Актирование.

Подразделения - на закладке "Подразделения" надо указать подразделения, по которым будут подбираться акты выполненных работ. Для каждого подразделения будет сформирован отдельный документ Актирование.

Далее необходимо нажать кнопку "Заполнить". В табличную часть попадут акты выполненых работ с состояниями:

  • Отправлен заказчику
  • Вручен заказчику
  • Подписан заказчиком

Попадут только те акты, которые еще не заактированы и имеют дату регистрации в текущем месяце.

Акты, отмеченные птичками, по кнопке "Загрузить" будут загружены в документы Актирование и проведены.

Отметки для загрузки не устанавливаются, если:

  • у этапа договора нет Планового срока
  • у этапа договора нет Дела
  • если условие в этапе договора Предоплата и процент оплаты акта меньше, чем в условии.
