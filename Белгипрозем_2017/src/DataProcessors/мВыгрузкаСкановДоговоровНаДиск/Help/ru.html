Обработка Выгрузка сканов договоров на диск проверяет наличие файлов во всех подкаталогах с именами районов в указанном каталоге. Если в подкаталогах есть файлы типа НННН-ГГ-Э.pdf, где

  • НННН - числовой номер дела/договора
  • ГГ - последние 2 цифры даты регистрации договора
  • Э - номер этапа договора, к которому относится дело,

  то обработка находит сканы договоров, соответствующих файлам отсканированных дел, и располагает их в тех же подкаталогах, где хранятся сканы дел.

Если договор в базе данных не найден или у него нет скана, то после обработки будет выдана информация об этом, которую следует передать в плановый отдел для его создания/сканирования.
