Обработка предназначена для создания однотипных исходящих писем (писем с одинаковым содержанием и темой) разным контрагентам.

В поле Шаблон письма надо выбрать вордовский шаблон письма, который будет заполняться необходимым содержимым.

На закладке Письма показана информация о подготовленных письмах.

На закладке Общие реквизиты необходимо заполнить все поля, которые потребуются для создания писем.

Поле Комментарий служит для определения писем, которые отправляются данной рассылкой. При повторном заполнении обработки, если будут письма с таким же комментарием, то они подставятся в список, иначе будут созданы новые. 

Порядок работы

По кнопке Заполнить закладка Письма заполняется корреспондентами юрлицами, у которых есть незакрытые этапы договоров, за исключением этапов с видом работы Оказание услуг по обращению за государственной регистрацией.

В письмах подставляется почтовый адрес корреспондента. Если почтовый адрес не указан, тогда берется юридический адрес.

Перед тем, как нажать Создать письма, надо заполнить поля на закладке Общие реквизиты.

По команде Создать письма будут созданы новые исходящие письма, к ним будут прикреплены созданые печатные формы писем. Если исходящее письмо есть, то новое создаваться не будет, печатная форма будет прикреплена к существующему. 

По команде Печать будут выведены на просмотр печатные формы писем.
