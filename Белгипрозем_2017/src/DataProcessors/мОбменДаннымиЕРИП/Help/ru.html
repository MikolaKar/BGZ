Обработка "Обмен данными ЕРИП" позволяет обмениваться файлами с данными с ЕРИП. 

Для использования возможностей обмена данными с ЕРИП необходимо включить функциональную опцию "Использовать обмен ЕРИП".

Надо заполнить следующие константы:

  • Каталог обмена с ЕРИП - путь к каталогу с файлами обмена с ЕРИП;
  • Код абонента для ЕРИП - код, выданный ЕРИП при регистрации организации для обмена данными;
  • Номер услуги для ЕРИП - номер услги, присвоенный в ЕРИП для проведения оплат.

 

Описание работы с подсистемой

Определение договоров для передачи в ЕРИП

Для выгрузки договоров в ЕРИП надо в форме договора после его записи нажать кнопку "ЕРИП". В форме списка догворов надо выделить один или несколько необходимых для передачи договоров и нажать кнопку "ЕРИП". Выделенные договора будут зарегистрированы для передачи в ЕРИП.

Передача файлов в ЕРИП

Для передачи файлов надо открыть данную обработку из меню Договоры - Сервис - Обмен данными с ЕРИП. В обработке доступны 2 кнопки:

  • Выгрузить - выгружает информацию из договоров, зарегистрированных для передачи в ЕРИП, в файл согласованного с ЕРИП формата с расширением *.202.
  • Проверить ответы - читает файлы, ответы от ЕРИП, с расширением *.204, на наличие ошибок в выгруженных файлах.

Загрузка файлов из ЕРИП

Загрузка файлов из ЕРИП выполняется из формы документа "Реестр текущих платежей" по кнопке "Загрузить из ЕРИП".

В открывшемся окне данной обработки надо указать файл, который будет загружен и нажать кнопку "Загрузить". Данные из файла будут добавлены в табличные части документа. При загрузке происходит автоматический поиск договора по его номеру, проверяется совпадение ФИО заказчика в договоре и в файле. При несовпадении ФИО или не нахождении договора поле Договор остается пустым и выдается сообщение о том, что есть  ошибка в при загрузке строки файла с указанием номера договора и ФИО.

Файл не будет загружен, если дата документа Реестр текущих платежей и дата поступления денежных средств на расчетный счет предприятия, указанная в загружаемом файле будут различаться.

Контроль состояния обмена с ЕРИП

Узнать в каком состоянии находится договор, переданный в ЕРИП, можно по команде Обмен с ЕРИП из формы договора.

Могут быть следующие состояния:

  • Для отправки - Договор зарегистрирован для отправки, но еще не передан в ЕРИП;
  • Отправлен - Информация для оплаты передана в ЕРИП, но еще не подтверждено ее получение;
  • Ошибка - Информация по договору не была принята в ЕРИП из-за ошибки в файле передачи;
  • Ожидание оплаты - Информация была передана и успешно размещена на сервере ЕРИП для оплаты;
  • Оплачен - По договору поступила оплата
