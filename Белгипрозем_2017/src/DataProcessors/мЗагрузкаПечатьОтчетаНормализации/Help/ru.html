Обработка позволяет загрузить и распечатать данные отчета по нормализации границ.

При выборе Отчета таблица заполняется информацией из базы данных.

Действия команд:

  • Прочитать - читает данные из файла dbf-формата в таблицу формы.
  • Записать - записывает данные из таблицы в базу данных. Обновляет файл dbf-формата номером отчета. Прикрепляет файлы из каталога к отчету.
  • Печать - на основании таблицы создает и открывает на просмотр печатную форму отчета.
  • Сохранить - создает и прикрепляет к отчету подготовленную печатную форму отчета.  
  • Выгрузить - выгружает в указанный каталог файлы, прикрепленные к отчету. Также выгружается дополнительный файл формата DBF с имененем как номер отчета и в котором содержится информация из табличной части.
