Обработка предназначена для загрузки реализации журналов из файла, подготовленного в Бухгалтерии 7.7.

Заполнение полей

  • Актирование - выбрать отдельный документ Актирование, в который будут внесены созданные акты
  • Путь к файлу - путь к загружаемому файлу (*.xls)
  • Дата создания ЭСЧФ - дата, которой будут создаваться ЭСЧФ
  • Начальный номер ЭСЧФ - номер, с которого начнут формироваться ЭСЧФ
  • Организация - наша организация, от которой выставляются ЭСЧФ
  • Наш УНП, Наше название, Наш адрес - наши реквизиты, которыми будут заполняться ЭСЧФ

Порядок работы

По кнопке Прочитать файл данные из файла будут прочитаны в первую таблицу. В ней можно проверить загруженную информацию и отметить флажками нужные строки для формирования ЭСЧФ.

По кнопке Создать ЭСЧФ по отмеченным флажками строкам будут сформированы ЭСЧФ.

По кнопке Создать акты будет заполнен и проведен документ Актирование, указанный в шапке обработки. По загружаемым журналам, которые были отгружены безвозмездно (по приказу), внутренние документы вида Акт выполненных работ не создаются и в ЭСЧФ не заполняются.
