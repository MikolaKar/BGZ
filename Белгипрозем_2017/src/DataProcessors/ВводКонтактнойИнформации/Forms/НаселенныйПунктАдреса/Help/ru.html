Форма предназначена для ввода населенного пункта по детальному шаблону.

В поле "Регион" необходимо ввести название субъекта Российской Федерации, к которому относится населенный пункт. Поля "Район", "Город", "Нас. пункт" предназначены для ввода соответствующих частей адреса. Поля, которые должны быть заполнены, зависят от адреса. Например, в некоторых адресах может не быть района, в других - города или населенного пункта.

Если для региона адреса загружен адресный классификатор, то по мере набора в каждом поле будут предлагаться варианты соотвествующей части адреса с учетом полей, заполненных ранее.
Также, при загруженном классификаторе, можно выбрать часть адреса из списка, нажав на кнопку выбора.
При вводе поля "Регион", если адресный классификатор для этого субъекта Российской Федерации не заполнен и у текущего пользователя есть права на загрузку, будет выведено предложение загрузить классификатор.
