Обработка Выгрузка ЗП предназначена для выгрузки информации о начисленной заработной плате в программу Зарплата и Управление Персоналом 8.

Заполнение полей:

  • Файл выгрузки - месторасположение и имя файла (*.dbf), который затем должен быть передан.
  • Период - отчетный период, за который должны быть выбраны данные.
  • Подразделение - подразделение, по сотрудникам которого необходимо передать информацию. Если не выбрано, то выгрузка будет сформирована по всем подразделениям.
