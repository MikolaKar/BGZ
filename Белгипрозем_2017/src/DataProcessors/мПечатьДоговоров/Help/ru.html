Обработка "Печать договоров" позволяет вывести на печать заполненный данными шаблон договора.

Кнопка  позволяет обновить поле Настройка после внесения изменений в Настройках печати без перезапуска обработки.

Кнопка  Настройки печати позволяет перейти к справочнику Настройки печати договоров.

При установленном флажке Создать документ вместо вывода на печать будет создан и сохранен документ в файлах договора.

При выборе договора или открытии формы в поле Настройка показываются только те настройки, для которых в справочнике Настройки печати договоров, есть строки для вида документа договора.
