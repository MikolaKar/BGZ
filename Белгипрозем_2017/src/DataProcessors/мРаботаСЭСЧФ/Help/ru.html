Работа с ЭСЧФ позволяет создавать ЭСЧФ, отправлять их на портал МНС, получать статусы ЭСЧФ.

Настройка работы

  • Указать пути к каталогам:
  • Константа "Каталог схем проверки ЭСЧФ" - содержит путь к каталогу схем проверки ЭСЧФ, поставляемых порталом.
  • Константа "Каталог выгрузки ЭСЧФ" - содержит каталог для хранения файлов выгружаемых ЭСЧФ, приема ответов от портала. 
  • Выбрать Период - период регистрации Актов выполненных работ.
  • Заполнить Дату создания ЭСЧФ. По умолчанию - текущая дата.
  • Указать начальный номер ЭСЧФ. По умолчанию - номер на 1 больше максимального номера ЭСЧФ.

    • При открытии формы она автоматически заполняется информацией.

    Выставление исходных ЭСЧФ

    На закладке Акты для выставления находятся акты, имеющие состояние Вручен заказчику, Подписан заказчиком, в которых есть НДС, которые были зарегистрированы в выбранном Периоде и по которым еще не были выставлены ЭСЧФ.

    По кнопке Заполнить акты табличная часть будет перезаполнена новыми данными. 

    По кнопке Создать ЭСЧФ по актам, отмеченным флажками, будут созданы ЭСЧФ. Табличная часть будет очищена, а подготовленные ЭСЧФ будут находится на следующей закладке ЭСЧФ к отправке.

    На закладке ЭСЧФ к отправке по кнопке Отправить запустить процедуру электронной подписи и передачи ЭСЧФ на портал. ЭСЧФ, отмеченные флажками, будут выгружены на портал. Процесс сопровождается индикацией процента выполнения и информацией о статусе выгружаемых ЭСЧФ. Если по акту выполненных работ есть ЭСЧФ со статусом Выставлен, Выставлен.Подписан покупателем или На согласовании, то акт будет открываться только на просмотр.

    Для выгрузки на портал нескольких ЭСЧФ следует использовать кнопку Подобрать ЭСЧФ.

    В случае успешной отправки ЭСЧФ из табличной части удаляются и показываются на следующей закладке Выставленные. На этой закладке можно проверить статусы ЭСЧФ, отмеченных флажками, нажав на кнопку Обновить статусы.

    На закладке Аннулированные отображаются ЭСЧФ, имеющие статус Аннулирован.

    Выставление исправленных ЭСЧФ

    В случае неправильного выставления ЭСЧФ, необходимо:

    1. В ЭСЧФ установить флажок Разрешить изменение акта. Записать и закрыть ЭСЧФ.
    2. Изменить Акт выполненных работ так, что бы он отображал полные правильные данные. В случае, если надо Аннулировать ЭСЧФ полностью, т.е. Акт был сформирован ошибочно, в Акте надо указать нулевые показатели. 
    3. Добавить его вручную на закладке Акты для выставления. При этом должна заполниться информацией строка таблицы.
    4. По кнопке Создать исправленный ЭСЧФ, по отмеченным флажками актам будут созданы новые ЭСЧФ, которые отобразятся на закладке ЭСЧФ к отправке.
    5. Перейти на закладку ЭСЧФ к отправке и по кнопке Отправить запустить процедуру электронной подписи и передачи ЭСЧФ на портал. ЭСЧФ, отмеченные флажками, будут выгружены на портал. ЭСЧФ, который был неправильно выставлен раньше, на портале будет аннулирован.
