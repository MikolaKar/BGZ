Для создания сразу нескольких актов по разным этапам - выделите их в таблице. По умолчанию для создания актов помечаются только те этапы, которые еще не заактированы.

При печати из договора суммы берутся из этапов договора. Если печать выполняется из акта выполненных работ, то суммы берутся из акта, а не из этапа договора.
