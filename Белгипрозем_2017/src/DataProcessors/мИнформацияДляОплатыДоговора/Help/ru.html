Обработка "Информация для оплаты договора" предназначена для печати аналогов извещения на оплату платежей по договору подряда с физическим лицом.

В форме необходимо отметить какая информация должна быть распечатана:

  1. Для оплаты договора подряда
  2. Для оплаты госпошлины
  3. Для оплаты госрегистрации
  4. Для оплаты ограничений (обременений) прав.

Для открытия обработки разделе "Внутренние документы" выполните команду "Печать". В списке выберите "Информация для печати".

После отметки пунктов для печати выполните команду "Печать", расположенную в форме.
