Обработка предназначена для пакетного формирования исходящих писем, печати сопроводительных писем и уведомлений к ним.

Порядок работы

  1. Заполнить необходимые поля для отбора актов, на основании которых будут формироваться письма:
    • Период
    • Вид договора
    • Кто подготовил акты
    • Организация
  2. Заполнить информацию, которая будет вноситься в письма:
    • Шаблон - Шаблон письма, который будет использован для печати
    • Брать подписал из договора - при установленном флажке в письмах будет проставлен тот, кто подписал договор, иначе будет взят из поля Подписал.
  3. Нажать кнопку Заполнить акты. В результате таблицы будут заполнены актами, которые соответствуют заданным условиям отбора. В таблицы не попадут акты, у которых есть отметка в журале передачи. Если есть нескольколько актов по разным договорам,но с одним корреспондентом, то они попадут в одно исходящее письмо.
  4. По кнопке Проверить проверяется наличие адреса и длина наименования корреспондента и адреса. Поля с ошибками отмечаются указателем.
  5. По кнопке Создать исходящие письма будут созданы исходящие документы и к каждому будет прикреплена печатная форма сопроводительного письма. В колонках Исх письмо и Карточка файла будут размещены на них ссылки.
  6. По кнопке Печать сопроводительных отмеченные флажками письма будут выведены на печать.
  7. По кнопке Печать уведомлений, для отмеченных флажками писем будут сформированы уведомления. Уведомления надо печатать на 2-х сторонах листа с поворотом по короткой стороне. На одном листе размещается 4 уведомления, которые после печати надо разрезать на 4 части по линиям.
