Обработка  позволяет выполнять определенные действия при сканировании документов в зависимости от выбранной операции.

Регистрация даты дела - При сканировании титульного листа дела автоматически устанавливается текущая дата как дата регистрации дела.

Прием дела в подразделение - При сканировании титульного листа у дела автоматически устанавливается состояние "Получено на проверку".

Прием дела на проверку - При сканировании титульного листа дела автоматически устанавливается состояние "На проверке". Если Акта проверки нет, то он создается и открывается. Если Акт проверки есть, то он открывается для редактирования. Если актов несколько - то открывается список для выбора акта для редактирования.

Акты помеченные на удаление не показываются.
