Регистр "Даты договоров" предназначен для хранения информации в разрезе Контрагентов, Договоров, Этапов работ по различным видам дат.

Важно! Если Вид даты договора это Дата подписания заказчиком, то в такой записи нельзя изменять Период и Вид даты договора. При необходимости изменения Периода надо скопировать эту запись в новую, а затем удалить. Дальше работать с новой записью.
