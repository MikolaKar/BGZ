Для открытия списка индексов следует открыть раздел "Нормативно-справочная информация" и с помощью команды панели навигации "Индексы нумерации" открыть форму списка индексов.

Индекс представляет собой произвольную цифро-буквенную строку размером до 5-ти символом. Значение индекса будет добавляться к номеру документа.

Индексы нумерации используются нумераторами при формировании регистрационных номеров для входящих, исходящих и внутренних документов, а также корреспондентов, организации и подразделений, вопросов деятельности.

Информация об индексе вводится в отдельной форме. Для создания индекса используйте команду "Создать", для редактирования - команду "Изменить".

Для нового индекса следует указать объект, для которого формируется индекс. Сначала следует выбрать вид объекта, а затем указать конкретный объект. В поле "Индекс" вводится собственно значение индекса. В поле "Комментарий" вводится пояснительный текст к индексу (например, по его назначению).
