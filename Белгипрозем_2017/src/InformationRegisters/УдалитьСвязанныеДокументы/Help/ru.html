В некоторых случаях необходимо установить связь различных документов. Для этого в карточке документа необходимо перейти на закладку "Связи", которая показывается, если настройка "Использовать связанные документы" установлена.

Для добавления связанного документа следует использовать команду "Создать", для изменения - команду "Изменить" списка "Ссылается на документы". На экран выводится форма настройки связи. В форме следует в поле "Ссылается на:" указать документ, на который ссылается текущий документ. В поле "Комментарий" можно ввести описание данной связи.

В списке может быть указано несколько связанных документов.

Если открыть карточку документа, на который ссылается текущий документ (нужно открыть форму связи и в поле "Ссылается на:" нажать кнопку открытия), то в списке "Ссылки из документов" показывается текущий документ, а также другие документы (если такие есть), в которых установлена связь этим документом. Для открытия карточки документа, из которого ссылаются, достаточно в списке выбрать строку с этим документом и дважды щелкнуть ее мышью (нажать клавишу Enter).
