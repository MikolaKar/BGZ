Регистр сведений "Параметры видов работ по договорам" служит для хранения дополнительной информации по видам работ.

Для просмотра параметров вида работы по договору следует открыть в разделе "Договора" в списке "Виды работ для физ лиц" или "Виды работ для юр лиц" необходимый элемент. Затем в панелени навигации этого элемента выбрать "Параметры видов работ по договорам".

Для вида работ следует задать:

  • Период - дата, с которой будут применяться устанавливаемые параметры;
  • Вид работ - ссылка на вид работ, к которому относятся параметры
  • Пред условие - текст, который будет печататься в договоре перед количеством дней выполнения работ. Например: не более.
  • Пост условие - текст, который будет печататься в договоре после количества дней выполнения работ.
  • Единица измерения - текст, который будет печататься в акте выполненных работ в графе "Единица измерения".
  • Вид дней - "Рабочие" либо "Календарные". Параметр используется как для печати в договоре, такт и при определении планового срока окончания работ.
  • Дней по городу, Дней по области - количество дней выполнения работ. Нужное значение будет выбрано в соответствии с указанным населенным пункте в месторасположении "Объекта работ". Если название населенного пункта совпадает с названием населенного пункта в константе "Основной город", то будут взяты для расчета планового срока и печати в договоре дни по городу, иначе - по области. Отдельно, Совместно - позволяют задать разное количество дней в зависимости от того, выполняется данный вид работ отдельно либо совместно с другим. Задается установкой/снятием флажка "Порядок выполнения совместно с другим этапом" в справочнике "Этапы договоров".
  • Землепользователи - при установке флажка Больше 5 можно задать дополнительное количество дней выполнения работ.
  • Учет дат - для расчета планового срока будет выбрана максимальная дата из отмеченных флажками: 
    • Подписания договора - дата подписания договора обеими сторонами;
    • Предоставления пакета документов - дата предоставления необходимого для выполнения работ пакета документов;
    • Возврата договора - дата возврата договора от Заказчика;  
    • Предоплаты - дата поступления предоплаты. Размер предоплаты оговаривается в справочнике "Этапы договоров";
    • Выполнения предыдущего этапа - дата выполнения работ по предыдущему этапу.
