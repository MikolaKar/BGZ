В списке Все правила автозапуска процессов можно настроить правило для автоматического старта процессов по заранее настроенному шаблону. Например: можно сделать так, чтобы зарегистрированная жалоба сразу отправилась на рассмотрение ответственному лицу.

Важно учесть, что:

  • Надо заранее создать шаблон процесса и привязать его к виду документов
  • Вид документа правила должен соответствовать виду документа, к которому привязан шаблон процесса
  • Организация, указанная в правиле должна соответствовать организации шаблона процесса
  • В шаблоне процесса должны быть заполнены все обязательные поля для старта

Как настроить привязку к виду документов в шаблоне процессов

Чтобы настроить правило автозапуска процессов, необходимо, чтобы вид документа в шаблоне процесса совпадал с видом документа правила автостарта.

Для этого:

  1. Перейдите в раздел Настройка и администрирование
  2. Выполните команду Шаблоны процессов панели навигации
  3. Щелчком мыши выберите процесс из списка
  4. Выполните двойной щелчок мыши по нужному шаблону, в новом окне откроется карточка шаблона
  5. Выполните команду Назначен видам документов панели навигации
  6. Двойным щелчком мыши выберите вид документа, если он уже есть в списке Кому назначен или создайте новый, нажав на кнопку Создать. В обоих случаях откроется карточка настройки шаблона процесса
  7. Заполните все необходимые поля, нажмите на кнопку Записать и закрыть, чтобы сохранить настройку шаблона процесса.

Как создать новое правило автозапуска процессов

Чтобы настроить новое правило автозапуска процесса из списка Все правила автозапуска процессов:

  1. Нажмите на кнопку Создать, в новом окне откроется карточка создания правила
  2. Выберите Вид бизнес-события из списка или введите с клавиатуры вручную
  3. Выберите Класс источника бизнес-события из списка
  4. В новом окне выберите нужный вид документа
  5. Выберите Организацию источника бизнес-события из списка или введите с клавиатуры вручную
  6. Щелчком мыши выберите шаблон процесса из списка
  7. В открывшемся окне выберите соответствующий шаблон процесса
  8. Нажмите на кнопку Записать и закрыть, чтобы сохранить правило

Общие сведения

Автостарт процессов является частью механизма бизнес-событий, который обеспечивает взаимодействие между другими механизмами программы. Например, между механизмом учета документов и механизмом бизнес-процессов. Он позволяет настроить автостарт процессов при возникновении тех или иных событий, указанных в заранее заполненных шаблонах.
