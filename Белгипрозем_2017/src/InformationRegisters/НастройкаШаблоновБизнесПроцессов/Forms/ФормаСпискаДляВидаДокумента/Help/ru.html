Список назначенных шаблонов для видов документов открывается из карточки вида документов с помощью команды панели навигации "Шаблоны процессов". 

В форме настраивается использование шаблонов для видов документов.

Для просмотра информации о настройке следует открыть форму настройки шаблона процесса.

Для получения информации об использовании шаблонов следует использовать отчет Шаблоны процессов.
